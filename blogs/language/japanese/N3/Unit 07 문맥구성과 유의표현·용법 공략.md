## 개요

N3 文字·語彙(문자·어휘) 후반부의 **文脈規定(문맥규정)**, **言い換え類義(유의표현)**, **用法(용법)** 세 영역은 어휘량 + 콜로케이션(共起·연어) 감각으로 승부한다. 단어를 외울 때 **함께 쓰이는 동사·조사·문형**까지 함께 외우는 것이 정답.

## 핵심 문법

### 1. 문맥규정(文脈規定) 공략

문장 빈칸에 가장 자연스러운 어휘를 고르는 문제. **콜로케이션**이 1순위.

- 「会議に___」 → 出席(しゅっせき)する / 参加する (○) vs 出会う (×)
- 「夢を___」 → 見る / 叶(かな)える (○) vs する (×)
- 「責任を___」 → 持つ / 取る / 果たす (○) vs 作る (×)
- 「興味を___」 → 持つ / 失(うしな)う (○) vs する (×)

### 2. 유의표현(言い換え類義) 공략

- やっと = ようやく (마침내)
- いつのまにか = 気がついたら (어느새인가)
- たぶん = おそらく (아마)
- 主に = ほとんど ~は (주로)
- いきなり = 突然(とつぜん) (갑자기)
- だんだん = 次第(しだい)に (점점)
- とうとう = ついに (드디어)
- 必ず = きっと (반드시)
- すぐに = ただちに (즉시)
- 全部 = すべて (전부)

### 3. 용법(用法) 공략

같은 단어가 여러 문장에서 자연스럽게 쓰이는 경우를 고른다. **품사·문형 위치·의미 범위**를 점검.

- 「すっきり」(개운한·산뜻한): 「気分が**すっきり**した」 ○ / 「料理が**すっきり**できた」 ×
- 「うっかり」(깜빡): 「**うっかり**忘れた」 ○ / 「**うっかり**好きだ」 ×
- 「思いつく」(생각나다·떠오르다): 「良いアイデアを**思いついた**」 ○ / 「日本語を**思いつく**」 ×

### 4. N3 핵심 콜로케이션 (자주 묻는 짝)

| 명사 | 어울리는 동사 |
|---|---|
| 約束(やくそく) | を守る / を破る / をする |
| 計画 | を立てる / を進める / を変える |
| 写真 | を撮(と)る / が写(うつ)る |
| 電話 | をかける / に出る / を切る |
| 心配 | をかける / をする / が消える |
| 試験 | を受ける / に合格(ごうかく)する / に落ちる |
| 風邪(かぜ) | をひく / が治(なお)る |
| お金 | をためる / を使う / を払(はら)う / を借りる |

### 5. 헷갈리는 유의어 묶음

- **見る vs 見える vs 見せる**: 보다 / 보이다(자동) / 보여주다(타동)
- **聞く vs 聞こえる vs 聞かせる**: 듣다 / 들리다(자동) / 들려주다(타동)
- **知る vs 分かる**: 정보 입수 / 이해
- **思う vs 考える**: 직관·감정 / 논리적 사고
- **始(はじ)める vs 始まる**: 시작하다(타) / 시작되다(자)

## JLPT N3 빈출 패턴

- 「~を取る」: 連絡, 写真, メモ, 休み, 資格, 注文, 責任
- 「~を出す」: スピード, 元気, 結論, 手紙, 宿題
- 「~がかかる」: お金, 時間, 電話, 心配, 迷惑(めいわく)
- 「お~する」/「ご~する」: 겸양표현 (Unit 12 참고)

## 예문

1. 「やっと宿題が終わった」 = **ようやく**宿題が終わった。 — 마침내 숙제가 끝났다.
2. 風邪を**ひいて**、会社を**休んだ**。 — 감기에 걸려 회사를 쉬었다.
3. 計画を**立てて**から**進める**ことが大事だ。 — 계획을 세운 뒤에 진행하는 것이 중요하다.
4. **うっかり**約束を忘れて、彼を怒(おこ)らせてしまった。 — 깜빡 약속을 잊어서 그를 화나게 했다.

## 연습 문제

1. 約束を___べきだ。
   (A) 守る  (B) 立てる  (C) 取る  (D) 出す

2. 「いきなり」と意味が最も近いのは?
   (A) ゆっくり  (B) 突然  (C) 必ず  (D) いつも

3. 「うっかり」の用法として最も自然なのは?
   (A) うっかり日本語が話せる
   (B) うっかり傘を電車に忘れた
   (C) うっかり花が咲いた
   (D) うっかり料理がおいしい

**정답**

1. (A) 守る — 약속을 「지키다」가 표준
2. (B) 突然 — 둘 다 「갑자기」
3. (B) — 「うっかり」는 부주의로 인한 실수에 사용
