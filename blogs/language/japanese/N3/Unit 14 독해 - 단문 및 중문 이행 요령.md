## 개요

N3 読解(독해)의 첫 관문, **단문(150~200자)**과 **중문(약 350자)**. 문제 유형은 ① 내용 일치, ② 필자의 의견·이유, ③ 밑줄 부분의 의미. 짧은 글일수록 **접속사·지시어·문말 표현**이 정답 단서다.

## 핵심 문법

### 1. 출제 패턴

| 영역 | 글자수 | 문제 수 | 시간 배분 |
|---|---|---|---|
| 단문 | 150~200자 | 4문제 (각 1문) | 글당 약 3분 |
| 중문 | 350자 전후 | 6문제 (글당 2문) | 글당 약 5분 |

### 2. 단문 풀이 전략

1. **질문 먼저 본다** — 무엇을 찾을지 결정
2. **첫 문장과 마지막 문장**에 결론·필자 의도가 압축됨
3. 지시어(これ·それ·そう) 등장 시 **앞 문장**을 다시 확인
4. 「のだ / だろう / と思う」 같은 **문말 표현**으로 주관·객관 판별

### 3. 중문 풀이 전략

1. **단락마다 1문장 요약** (속독)
2. **접속사(しかし·だから·つまり·一方·ところで)** 가 흐름 전환 신호
3. 「~とは / ~ということは」 = **정의·핵심 문장** 표시
4. 선택지 4개 중 2개는 본문 표현 변형 — 본문 그대로 vs 「~いう意味だ」 유의

### 4. 핵심 「독해 키워드」 정리

| 종류 | 표현 | 신호 |
|---|---|---|
| 결론·요약 | つまり / 要するに(ようするに) / このように | 정답 직전 신호 |
| 역접·반전 | しかし / ところが / だが / 一方 | 주장 방향 전환 |
| 인과 | だから / そのため / その結果 | 원인→결과 흐름 |
| 예시 | 例えば(たとえば) / 具体的には | 부차 정보 |
| 강조 | むしろ / 実は(じつは) / 確かに | 필자의 진의 |

### 5. 자주 묻는 「밑줄 친 ~は何を意味するか」 유형

- 「これ / それ / そう」가 가리키는 내용 → 직전 문장
- 추상명사(「問題」「変化」「課題」) → 단락 주제
- 비유·관용 표현 → 마지막 단락의 정리 문장

## JLPT N3 빈출 패턴

- 「**つまり**~ということだ」 = 결론 신호
- 「**しかし**~」 = 정답 단락 가능성↑
- 질문이 「**筆者は何が言いたいのか**」면 → 마지막 단락 1·2문장
- 질문이 「**~とあるが、それはなぜか**」면 → 밑줄 직후 또는 다음 문장

## 예문 — 단문 샘플

**다음 글을 읽고 질문에 답하시오.**

> 最近、自宅で仕事をする人が増えている。通勤時間がなくなり、自分の時間を有効に使えるという利点がある。しかし、家族や仕事の境界(きょうかい)があいまいになり、集中(しゅうちゅう)できないという声も多い。在宅勤務(ざいたくきんむ)の良さを生かすためには、自分なりのルールを作ることが大切だ。

**질문**: 筆者が一番言いたいことは何か。

- (A) 在宅勤務は時間の節約(せつやく)になる。
- (B) 在宅勤務にはルールが必要だ。
- (C) 在宅勤務は集中できないので避けるべきだ。
- (D) 家族との時間を増やすべきだ。

**정답**: (B) — 마지막 문장이 결론, 「ルールを作ることが大切だ」

## 연습 문제

**다음 글을 읽고 답하시오.**

> 子供のころ、私は野菜が大嫌(きら)いだった。母はいつも「体にいいから食べなさい」と言ったが、なかなか口に入らなかった。ところが、大人になって自分で料理を作るようになってから、野菜の美味しさが分かるようになった。**自分で作る**と、味だけでなく、苦労(くろう)も分かるようになるのかもしれない。

1. 筆者が野菜を食べるようになったきっかけは何か。
   (A) 母にしかられたから
   (B) 体にいいと知ったから
   (C) 自分で料理を作るようになったから
   (D) 野菜の苦労を知ったから

2. 下線「自分で作る」の指す意味として最も近いのは?
   (A) お金を稼ぐ  (B) 体を動かす  (C) 料理を作る  (D) 野菜を育てる

3. 筆者が一番言いたいことは何か。
   (A) 子供のころから野菜を食べるべきだ
   (B) 母の言葉は正しかった
   (C) 自分で作ることで物事の良さを知ることができる
   (D) 大人になると味覚(みかく)が変わる

**정답**

1. (C) — 「自分で料理を作るようになってから」
2. (C) — 직전 문맥에서 「料理を作る」
3. (C) — 「自分で作ると~分かるようになる」 결론
