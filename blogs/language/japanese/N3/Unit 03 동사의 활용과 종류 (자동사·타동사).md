## 개요

동사 활용은 N3 문법의 척추(背骨). 1·2·3그룹 분류, ます形·て形·ない形·辞書形·た形·命令形·意向形을 자유롭게 오갈 수 있어야 한다. 추가로 N3에서 본격 등장하는 自動詞(자동사)·他動詞(타동사) 짝짓기가 핵심.

## 핵심 문법

### 1. 동사 3그룹 분류

| 그룹 | 특징 | 예 |
|---|---|---|
| 1그룹 (五段動詞) | 어미가 う단(う·く·ぐ·す·つ·ぬ·ぶ·む·る) | 書(か)く, 話す, 飲む, 帰(かえ)る |
| 2그룹 (一段動詞) | る로 끝나고 앞이 い단/え단 | 食べる, 見る, 起きる, 教える |
| 3그루 (変格動詞) | 불규칙 | する, 来(く)る |

- 외형은 2그룹이지만 1그룹인 함정 동사: 帰る, 入る, 走る, 知る, 切る — 모두 1그룹

### 2. 주요 활용형

| 활용 | 1그룹 (書く) | 2그룹 (食べる) | 3그룹 (する/来る) |
|---|---|---|---|
| ます形 | 書きます | 食べます | します / 来(き)ます |
| て形 | 書いて | 食べて | して / 来(き)て |
| ない形 | 書かない | 食べない | しない / 来(こ)ない |
| 辞書形 | 書く | 食べる | する / 来る |
| た形 | 書いた | 食べた | した / 来た |
| 가능형 | 書ける | 食べられる | できる / 来られる |
| 의지·권유 | 書こう | 食べよう | しよう / 来よう |
| 명령형 | 書け | 食べろ | しろ / 来い |
| 조건형(ば) | 書けば | 食べれば | すれば / 来れば |

### 3. て形 만드는 법 (1그룹 핵심)

| 어미 | 변화 | 예 |
|---|---|---|
| う·つ·る | って | 買う→買って, 待つ→待って, 帰る→帰って |
| む·ぶ·ぬ | んで | 飲む→飲んで, 遊ぶ→遊んで, 死ぬ→死んで |
| く | いて | 書く→書いて (예외: 行く→行って) |
| ぐ | いで | 泳ぐ→泳いで |
| す | して | 話す→話して |

### 4. 自動詞(자동사)와 他動詞(타동사)

- **자동사**: が + 자동사 (주체의 동작·상태)
- **타동사**: を + 타동사 (대상에게 행하는 동작)

| 자동사 (~が) | 타동사 (~を) |
|---|---|
| ドアが開(あ)く | ドアを開(あ)ける |
| 電気がつく | 電気をつける |
| 窓(まど)が閉(し)まる | 窓を閉(し)める |
| 子供が起きる | 子供を起こす |
| 授業が始(はじ)まる | 授業を始(はじ)める |
| お湯が沸(わ)く | お湯を沸(わ)かす |
| 車が止(と)まる | 車を止(と)める |

### 5. ている vs てある

- **자동사 + ている**: 결과 상태 — 「ドアが開**いている**」(문이 열려 있다)
- **타동사 + てある**: 누군가의 의도로 그 상태 — 「ドアが開**けてある**」(누가 열어둔 상태)

## JLPT N3 빈출 패턴

- 「電気が___」 → ついている (자동사)
- 「電気を___」 → つけてある / つけておく (타동사)
- 帰る의 て形 → 帰**って** (1그룹)
- 行く의 て形 → 行**って** (예외)

## 예문

1. **窓**(まど)を**閉**(し)めて**ください**。 — 창문을 닫아 주세요. (타동사)
2. もう**会議**(かいぎ)は**始まっている**。 — 이미 회의는 시작되어 있다. (자동사+ている)
3. 子供を**起こさない**でください。 — 아이를 깨우지 마세요. (타동사 ない形)
4. お湯を**沸かして**、お茶を**入れた**。 — 물을 끓이고 차를 우렸다.

## 연습 문제

1. 雨が降っているので、傘を___ましょう。
   (A) 開く  (B) 開ける  (C) 開いて  (D) 開け

2. 「行く」의 て形은?
   (A) 行いて  (B) 行きて  (C) 行って  (D) 行くて

3. 「帰る」는 몇 그룹 동사?
   (A) 1그룹  (B) 2그룹  (C) 3그룹  (D) 불규칙

**정답**

1. (B) 開ける — 우산을(目的語) 펴다 → 타동사
2. (C) 行って — く로 끝나지만 예외적으로 「行って」
3. (A) 1그룹 — 외형은 2그룹 같지만 1그룹 함정 동사
