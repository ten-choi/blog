## 개요

N3 聴解(청해)의 1·2부 — **과제이해(課題理解)**와 **포인트이해(ポイント理解)**. 둘 다 「**누가 / 무엇을 / 언제 / 어디서 / 왜**」를 묻는다. 차이는 ① 과제이해 = **다음에 할 행동**, ② 포인트이해 = **이유·세부정보**.

## 핵심 문법

### 1. 출제 구성

| 영역 | 문제 수 | 길이 | 핵심 |
|---|---|---|---|
| 課題理解 | 6문제 | 짧은 대화 + 그림·문장 선택 | 「これからどうするか」 |
| ポイント理解 | 6문제 | 짧은 대화 | 「なぜ / 何を / どこで」 |

### 2. 과제이해 풀이 전략

1. **질문이 먼저 나온다** → 무엇을 듣는지 명확히
2. 대화 중 「**번복**」 자주 등장: 「**やっぱり / でも / じゃ / ~に変更します**」 → 마지막 결정이 정답
3. 등장인물이 둘 이상이면 **누구의 행동을 묻는지** 확인
4. 시간·순서 단어 주목: 「**まず / 先に / 後で / 最後に**」

### 3. 포인트이해 풀이 전략

1. 질문 후 **20초 정도 선택지 보는 시간** 제공 — 빠르게 키워드 점검
2. 「**理由 / 原因 / どうして**」 키워드 등장 시 「**~から / ~ので / だって**」 부분 집중
3. 부정문 함정: 「~ないわけではない」 = 약한 긍정
4. 「~たいけど…」 = 결국 못 한다 / 「~たかったが、~した」 = 다른 행동을 했다

### 4. 변경·번복 핵심 표현

- 「**やっぱり**~にします」 — 역시 ~로 할게요
- 「**じゃ**~にしましょう」 — 그럼 ~합시다
- 「**ちょっと変更(へんこう)して**」 — 좀 바꿔서
- 「**それより**~」 — 그보다는 ~
- 「**やっぱりやめて**~」 — 역시 그만두고 ~

### 5. 이유 묻기 빈출 표현

- 「**どうして**~ですか」 — 왜
- 「**なぜ**~したのですか」 — 어째서
- 「**~わけ**は?」 — ~한 까닭은?
- 답변에 「**実は**~/ **というのは**~」 → 진짜 이유 단서

### 6. 빈출 함정 — 음성 단서

- 숫자: 「**14日(じゅうよっか)**」 vs 「**40日(よんじゅうにち)**」 — 헷갈리는 발음
- 시간: 「**4時(よじ)**」 vs 「**7時(しちじ)**」
- 인원수: 「**4人(よにん)**」 vs 「**7人(しちにん)**」
- 가격: 「**1500円**」 vs 「**1万5千円**」 — 정확한 자릿수

## JLPT N3 빈출 패턴

- 회의실 예약: 누가, 몇 시, 몇 명 → 정확한 숫자 듣기
- 길 안내: 다음 행동 + 도착지 → 「**まず**~, **次に**~, **最後に**~」
- 약속 변경: 처음 약속 → 변경 → 최종 결정 → 항상 **마지막 결정**이 정답
- 도서·물건 부탁: 빌리는 사람과 빌려주는 사람 구분

## 예문 — 과제이해 샘플 (스크립트만 표시)

**상황**: 회사에서 남자와 여자가 회의 준비에 대해 이야기하고 있다.

> **男**: 明日の会議、資料(しりょう)はもう準備した?
> **女**: はい、コピーを20部用意しました。
> **男**: あ、すみません、参加者(さんかしゃ)が5人増えたんです。
> **女**: じゃ、あと5部追加(ついか)で印刷しますね。
> **男**: お願いします。それから、お茶も人数分(にんずうぶん)用意してください。
> **女**: 分かりました。コピーを先にしますね。

**질문**: 女の人はこの後、まず何をしますか。

- (A) お茶を用意する
- (B) コピーを5部追加で印刷する
- (C) 参加者を確認する
- (D) 資料を配(くば)る

**정답**: (B) — 「コピーを先にしますね」 = 우선 복사부터

## 연습 문제 (스크립트 기반)

**스크립트**: 学生と先生が話しています。

> **学生**: 先生、レポートの提出(ていしゅつ)、月曜日でしたよね。
> **先生**: あ、来週から学会(がっかい)で出張(しゅっちょう)するから、金曜日までに出してください。
> **学生**: 金曜日ですか。実は、テーマを変えたくて…。
> **先生**: テーマ変更(へんこう)するなら、来週の水曜日までで大丈夫です。

1. レポートの新しい提出日はいつですか。
   (A) 月曜日  (B) 金曜日  (C) 来週の水曜日  (D) 来週の月曜日

2. 学生はなぜ提出日が変わったのですか。
   (A) 先生が出張するから
   (B) テーマを変えたいから
   (C) 学会に参加するから
   (D) 資料を集めたいから

3. 先生は来週、何をしますか。
   (A) レポートを読む  (B) 学会に出る  (C) 授業をする  (D) 学生に会う

**정답**

1. (C) — 테마 변경하면 「来週の水曜日まで」
2. (B) — 학생이 「テーマを変えたくて」 → 결과로 제출일 변경
3. (B) — 「学会で出張する」
