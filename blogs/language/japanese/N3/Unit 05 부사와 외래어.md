## 개요

N3 부사는 **호응 관계**(공기·共起)와 함께 외운다. 「決して~ない」, 「たぶん~だろう」처럼 짝이 있는 부사가 빈출. 외래어(カタカナ)는 의미·표기를 함께 점검해야 한다.

## 핵심 문법

### 1. 빈도·정도 부사

| 의미 | 부사 |
|---|---|
| 항상·자주 | いつも, よく, たびたび, しょっちゅう |
| 가끔 | たまに, 時々(ときどき), ときには |
| 거의 ~않다 | あまり~ない, めったに~ない |
| 전혀 | 全然(ぜんぜん)~ない, 決して~ない |
| 매우 | とても, 非常(ひじょう)に, 大変 |
| 좀, 약간 | 少し, ちょっと, やや, わずかに |

### 2. 양태·확신 부사 (호응 표현)

| 부사 | 짝 (호응) | 의미 |
|---|---|---|
| たぶん | ~だろう / ~でしょう | 아마 ~일 것이다 |
| もしかしたら | ~かもしれない | 어쩌면 ~할지 모른다 |
| まるで | ~ようだ / みたい | 마치 ~같다 |
| ぜひ | ~たい / ~てほしい | 꼭 (희망·요청) |
| 決して | ~ない | 결코 ~않다 |
| ぜんぜん | ~ない | 전혀 ~않다 |
| まさか | ~とは思わなかった / ~ない | 설마 |
| きっと | ~だろう / 강한 추측 | 분명히 |

### 3. 상태·결과 부사 (오노마토페 포함)

- **ゆっくり**: 천천히
- **すっかり**: 완전히 (변화 결과) — 「すっかり忘れた」
- **しっかり**: 단단히, 야무지게
- **だんだん**: 점점
- **どんどん**: 자꾸자꾸, 점점 더
- **そろそろ**: 슬슬
- **やっと / ようやく**: 마침내

### 4. 시간·순서 부사

- 最初(さいしょ)に, 初(はじ)めに, まず — 처음에
- 次(つぎ)に, それから, そして — 다음에
- 最後(さいご)に, ついに, とうとう — 마지막에/마침내

### 5. N3 빈출 외래어(カタカナ)

| 카타카나 | 의미 |
|---|---|
| アイデア | 아이디어 |
| イメージ | 이미지 |
| エネルギー | 에너지 |
| カロリー | 칼로리 |
| サービス | 서비스 |
| スピーチ | 스피치 |
| データ | 데이터 |
| トラブル | 트러블, 문제 |
| バランス | 균형 |
| マナー | 매너, 예의 |
| メッセージ | 메시지 |
| リサイクル | 재활용 |
| レベル | 레벨 |

## JLPT N3 빈출 패턴

- 「たぶん明日は雨___」 → だろう/でしょう
- 「決して諦め___」 → ない
- 「すっかり___」 → 忘れる, 変わる, 良くなる
- 외래어 + する: アルバイト**する**, リサイクル**する**

## 예문

1. **たぶん**彼は来ない**でしょう**。 — 아마 그는 오지 않을 거예요.
2. **決して**あきらめ**ません**。 — 결코 포기하지 않습니다.
3. **そろそろ**帰りましょう。 — 슬슬 돌아갑시다.
4. このプロジェクトには新しい**アイデア**が必要だ。 — 이 프로젝트에는 새로운 아이디어가 필요하다.

## 연습 문제

1. ___彼女はもう知っているかもしれない。
   (A) 決して  (B) たぶん  (C) もしかしたら  (D) ぜひ

2. 約束を___忘れてしまった。
   (A) しっかり  (B) ゆっくり  (C) すっかり  (D) ぴったり

3. 健康のために、栄養の___が大事です。
   (A) サービス  (B) バランス  (C) メッセージ  (D) スピーチ

**정답**

1. (C) もしかしたら — 「かもしれない」와 호응
2. (C) すっかり — 「완전히 잊었다」 결과 강조
3. (B) バランス — 영양의 「밸런스(균형)」
