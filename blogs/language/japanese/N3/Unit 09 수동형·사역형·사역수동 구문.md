## 개요

受身(수동)·使役(사역)·使役受身(사역수동)는 N3 문법의 「3대 변환」. 동사 활용 + 조사(に·を) 변화 + 의미 뉘앙스(피해 수동 등)를 한 묶음으로 익혀야 한다.

## 핵심 문법

### 1. 수동형(受身) 만드는 법

| 그룹 | 변환 |
|---|---|
| 1그룹 | う단 → あ단 + れる: 書く → 書か**れる**, 飲む → 飲ま**れる** |
| 2그룹 | る 삭제 + られる: 食べる → 食べ**られる**, 見る → 見**られる** |
| 3그룹 | する → される, 来る → 来(こ)**られる** |

### 2. 수동의 기본 용법

- **직접 수동**: A는 B에게 ~당했다·받았다 — 「私は先生に**ほめられた**」(직접 칭찬받음)
- **평가·감정 수동**: 「**しかられる·怒られる·笑われる**」 — N3에서 가장 자주 등장
- **무생물 주어 수동**: 사물·사실 진술 — 「この本は多くの人**に読まれている**」
- N2 진학 시 「간접 수동(피해 수동)」과 「소유물 수동」의 미세 의미를 추가 학습

### 3. 사역형(使役) 만드는 법

| 그룹 | 변환 |
|---|---|
| 1그룹 | う단 → あ단 + せる: 書く → 書か**せる**, 飲む → 飲ま**せる** |
| 2그룹 | る 삭제 + させる: 食べる → 食べ**させる** |
| 3그룹 | する → させる, 来る → 来(こ)**させる** |

### 4. 사역의 의미와 조사

- **강제** (A**가** B**에게** ~시키다): 「先生は**学生に**漢字**を**書か**せた**」
- **허가·방임**: 「親は子供**を**遊ば**せる**」

자동사 사역: 「を」 사용 — 子供**を**泣**かせる**
타동사 사역: 「に」 사용 — 子供**に**本**を**読**ませる**

### 5. 사역수동(使役受身) 기본형

- 의미: 「(누가) 시켜서 ~하다」 = 본의 아니게 ~하다·하게 됨
- 만드는 법: **사역형 + られる**

| 그룹 | 변환 |
|---|---|
| 1그룹 | 書かせる → 書か**せられる** |
| 2그룹 | 食べさせる → 食べ**させられる** |
| 3그룹 | する → さ**せられる** / 来る → 来(こ)**させられる** |

- 「~せられる」의 회화체 단축형(行か**される**·待**たされる**)은 N2에서 본격 학습

## JLPT N3 빈출 패턴

- 「~に~を**~(さ)せる**」: 사역
- 「~に**~(ら)れる**」: 수동
- 「~に~を**~(さ)せられる**」: 사역수동
- 「**ほめられる / しかられる / 怒られる / 笑われる**」: 감정·평가 수동 빈출

## 예문

1. 田中さんは先生に**ほめられた**。 — 다나카씨는 선생님께 칭찬받았다. (직접 수동)
2. 弟は母に**しかられた**。 — 동생은 어머니께 혼났다. (감정 수동)
3. 先生は学生**に**漢字**を**書か**せた**。 — 선생님은 학생에게 한자를 쓰게 했다. (사역)
4. 私は母**に**毎朝早く起き**させられた**。 — 나는 어머니 때문에 매일 아침 일찍 일어났다. (사역수동)

## 연습 문제

1. 田中さんは部長に___て、うれしかった。
   (A) ほめた  (B) ほめられ  (C) ほめさせ  (D) ほめさせられ

2. 先生は学生に作文を___。
   (A) 書いた  (B) 書かれた  (C) 書かせた  (D) 書かれさせた

3. 子供のとき、毎日ピアノを___て大変だった。
   (A) 練習した  (B) 練習された  (C) 練習させた  (D) 練習させられ

**정답**

1. (B) ほめられ — 칭찬받음 (직접 수동)
2. (C) 書かせた — 선생님이 시킴 (사역)
3. (D) 練習させられ — 시켜서 어쩔 수 없이 했다 (사역수동)
