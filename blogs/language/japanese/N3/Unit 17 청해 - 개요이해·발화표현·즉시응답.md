## 개요

N3 聴解(청해) 후반 3영역. ① **概要理解(개요이해)**: 글의 「전체 취지」를 파악, ② **発話表現(발화표현)**: 그림 상황에 맞는 한마디 고르기, ③ **即時応答(즉시응답)**: 짧은 한마디 들려주고 적절한 대답 즉시 고르기. 짧고 빠른 영역들이라 「**평소 회화 패턴 암기**」가 곧 점수.

## 핵심 문법

### 1. 출제 구성

| 영역 | 문제 수 | 시간 | 특징 |
|---|---|---|---|
| 概要理解 | 3문제 | 약 30~50초 발화 | 선택지가 문제지에 없다 (음성으로) |
| 発話表現 | 4문제 | 그림 + 짧은 상황 설명 | 적절한 한마디 고르기 |
| 即時応答 | 9문제 | 한 문장 발화 | 즉시 적절한 응답 |

### 2. 개요이해(概要理解) 전략

1. 선택지는 음성으로 → 메모 추천. ①②③④의 키워드만 짧게
2. 「**つまり~ということです / だから~なのです**」 = 정답 단서
3. 강의·뉴스 형식이 많음 — **첫 1~2문장**에서 화제 제시
4. 결론 문장 「~**だと思います / べきです**」 → 화자 의견

### 3. 발화표현(発話表現) 전략

상황 설명을 듣고, 그림 속 인물이 「**무슨 말을 할 차례**」인지 판단.

- 「**お先に失礼します**」: 회사 먼저 퇴근할 때
- 「**ごちそうさまでした**」: 식사 후
- 「**ただいま**」「**お帰りなさい**」: 귀가 인사
- 「**お疲れさまでした**」: 노고를 위로
- 「**お邪魔(じゃま)します**」「**お邪魔(じゃま)しました**」: 남의 집 방문/돌아갈 때
- 「**遅(おく)れてすみません**」: 약속 늦었을 때
- 「**お久しぶりです**」: 오랜만에 만났을 때

### 4. 즉시응답(即時応答) 전략

발화 → 응답 패턴 매칭. 의문문 형태에 따라 답이 달라진다.

| 발화 | 자연스러운 응답 |
|---|---|
| 「~でいいですか」 | 「はい、それで結構(けっこう)です」 |
| 「~でもいいですか」(허락) | 「はい、どうぞ」「ええ、構(かま)いません」 |
| 「手伝(てつだ)いましょうか」 | 「ありがとう、助かります」「いえ、大丈夫です」 |
| 「~なら~でしょう」 | 「そうですね」「だと思います」 |
| 「~たら?」(제안) | 「そうですね、そうします」「いいですね」 |
| 「~てくれない?」(부탁) | 「いいよ」「ごめん、今ちょっと…」 |
| 「お疲れさま」 | 「お疲れさまでした」 |
| 「ごめんなさい」 | 「いえいえ / 大丈夫ですよ」 |

### 5. 관용·정형 표현 (꼭 외울 것)

- 「**かしこまりました**」 — 알겠습니다 (서비스업·비즈니스)
- 「**承知(しょうち)しました**」 — 알겠습니다 (비즈니스)
- 「**とんでもないです**」 — 천만에요 / 별 말씀을
- 「**お言葉に甘えて**」 — 호의를 받아들여
- 「**遠慮(えんりょ)なく**~」 — 사양 말고 ~
- 「**そうですね、ちょっと…**」 — 거절의 완곡 표현
- 「**せっかくですが**」 — 모처럼이지만 (거절 도입)

### 6. 함정 패턴

- 「~ましたか」질문에 「**いいえ、まだです**」가 정답일 때 많음
- 「~た**ばかり**ですか」 → 「**はい、今**~」 / 「いいえ、もう~」
- 「~に**よろしく**」 → 「**はい、伝(つた)えます**」
- 부정 의문문 「~じゃないですか」 → 한국어와 반대 응답에 주의

## JLPT N3 빈출 패턴

- **개요이해**: 「全体として何について話していますか」 → 첫 2~3문장 핵심어
- **발화표현**: 사진·그림이 비즈니스 / 가정 / 식당 / 길 등 — 상황 단어 즉시 매칭
- **즉시응답**: 부탁·제안·허락·인사 — 정형 응답 9할
- 「**~てもらえますか / ~ていただけますか**」 → 「**はい、いいですよ / もちろんです**」

## 예문 — 발화표현·즉시응답 샘플

### 발화표현

**상황**: 동료가 야근하고 있는데, 본인만 먼저 퇴근하려 한다. 무슨 말을 할까?

- (A) **お疲(つか)れさまでした**
- (B) **お先(さき)に失礼(しつれい)します**
- (C) **おはようございます**
- (D) **ただいま**

**정답**: (B)

### 즉시응답

**발화**: 「すみません、この席、空いていますか。」

- (A) はい、どうぞ。
- (B) いいえ、もう座っています。
- (C) すみません、知りません。
- (D) はい、空き家(あきや)です。

**정답**: (A) — 자리 비었는지 물음에 「네, 앉으세요」

## 연습 문제

1. **발화표현**: 친구의 가게에서 식사를 마치고 나갈 때 무슨 말을 할까?
   (A) いただきます  (B) ごちそうさまでした  (C) おやすみなさい  (D) お疲れさまでした

2. **즉시응답**: 「手伝いましょうか。」
   (A) はい、お願いします。
   (B) いいえ、行きません。
   (C) はい、行きます。
   (D) ありがとう、来ました。

3. **즉시응답**: 「先生、明日もう一度伺ってもよろしいですか。」
   (A) はい、ご覧ください。  (B) ええ、いつでもどうぞ。  (C) いいえ、行きません。  (D) ご遠慮ください。

**정답**

1. (B) ごちそうさまでした — 식사 후 인사
2. (A) はい、お願いします — 도움 제안에 수락
3. (B) — 「방문해도 될까요」에 대한 정중한 허락 응답
