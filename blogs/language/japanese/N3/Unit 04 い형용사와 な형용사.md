## 개요

일본어 형용사는 い형용사(イ形容詞)와 な형용사(ナ形容詞) 두 종류. N3에서는 단순 활용을 넘어 て形 접속, 부사화(-く / -に), 명사화(-さ / -み)까지 확장된다.

## 핵심 문법

### 1. い형용사 vs な형용사

| 구분 | い형용사 | な형용사 |
|---|---|---|
| 어미 | い | (사전형은 단독, 명사 수식 시 な) |
| 명사 수식 | 高**い**山 | 静(しず)か**な**部屋 |
| 술어 | 高**いです** | 静(しず)か**です** |
| 과거 | 高**かった** | 静(しず)か**だった** |
| 부정 | 高**くない** | 静(しず)か**ではない** |
| 부정 과거 | 高**くなかった** | 静(しず)か**ではなかった** |

- 함정 な형용사 (い로 끝나지만 な형용사): きれい, 嫌(きら)い, 有名, 親切(しんせつ), 元気, 便利(べんり)

### 2. て形(연결)

| い형용사 | い → くて | 高い → 高**くて** |
| な형용사 | 어간 + で | 静か → 静か**で** |

- 「**安くて、おいしい**」 (싸고 맛있다)
- 「**静かで、便利な**ところ」 (조용하고 편리한 곳)

### 3. 부사화

- い형용사: い → **く** — 早い → 早**く** (빨리)
- な형용사: 어간 + **に** — 静か → 静か**に** (조용히)
- 예: 「子供が**早く**起きた」, 「**静かに**話して」

### 4. 명사화

- **-さ** (객관적 정도): 高い → 高**さ**, 重い → 重**さ**, 便利 → 便利**さ**
- **-み** (주관·감각): 重い → 重**み**, 苦(にが)い → 苦**み**, 楽し → 楽し**み**

### 5. N3 빈출 형용사

| い형용사 | な형용사 |
|---|---|
| 厳(きび)しい, 親(した)しい, 詳(くわ)しい | 真剣(しんけん)な, 自由な, 確(たし)かな |
| 偉(えら)い, 涼(すず)しい, 怖(こわ)い | 平気(へいき)な, 失礼な, 不安な |
| 苦(にが)い, 浅(あさ)い, 太(ふと)い | 様々(さまざま)な, 主(おも)な, 適切な |

## JLPT N3 빈출 패턴

- 「~く + 동사」 → 부사용법 (早く, 多く, 美しく)
- 「~なる」/「~する」: 강く**なる** (강해지다), 静か**になる** (조용해지다)
- 「~がる」: 第三者의 감정 — 寒**がる**, 嬉し**がる**
- 「~そうだ」(様態): 美味し**そう**だ, 元気**そう**だ

## 예문

1. この本は**面白くて、わかりやすい**。 — 이 책은 재밌고 이해하기 쉽다.
2. 部屋を**静かに**してください。 — 방을 조용히 해주세요.
3. **健康**(けんこう)の**大切**(たいせつ)**さ**に気づいた。 — 건강의 소중함을 깨달았다.
4. 田中さんは寒**がっている**ね。 — 다나카씨는 추워하고 있네.

## 연습 문제

1. このカメラは便利___、軽い。
   (A) て  (B) で  (C) に  (D) と

2. 朝、___起きてください。
   (A) 早い  (B) 早く  (C) 早いに  (D) 早くに

3. ___は時々失敗します。
   (A) 有名  (B) 有名な人  (C) 有名い人  (D) 有名で人

**정답**

1. (B) で — な형용사의 て형은 「で」
2. (B) 早く — い형용사의 부사화: い→く
3. (B) 有名な人 — な형용사가 명사 수식 시 「な」
