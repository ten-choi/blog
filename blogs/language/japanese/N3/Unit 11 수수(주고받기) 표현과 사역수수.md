## 개요

授受(수수)·やりもらい(주고받기) 표현은 일본어 사회성·인간관계 묘사의 뼈대. 「누가 누구에게」 + 「상하·내외 관계」를 정확히 파악해야 한다. N3에서는 단순 물건 주고받기 + **て形 + あげる/くれる/もらう** + **사역수수**까지 묶어 출제.

## 핵심 문법

### 1. 기본 수수 동사

| 동사 | 방향 | 주어→대상 |
|---|---|---|
| あげる / さしあげる | 내(우리) → 남 (남↑일 때 さしあげる) | 私→他人 |
| くれる / くださる | 남 → 나(우리) (남↑일 때 くださる) | 他人→私 |
| もらう / いただく | 나(우리)가 남에게서 받음 (남↑일 때 いただく) | 私が他人から |

### 2. 인물 관계 (内·外 / 상·하)

- 「内」(우치): 가족·소속 집단 (나의 영역) — くれる
- 「外」(소토): 외부 사람 — あげる/もらう
- 윗사람 ↑ : さしあげる / くださる / いただく
- 친한 친구·아랫사람 : やる (가족 안에서 아이에게도 종종)

### 3. て形 + 授受 (행위의 주고받기)

| 형태 | 의미 |
|---|---|
| ~て**あげる** | 내가 남에게 ~해주다 |
| ~て**くれる** | 남이 나에게 ~해주다 |
| ~て**もらう** | 내가 남에게 ~해 받다 (의뢰·은혜) |
| ~て**やる** | 손아랫사람·동물에게 |
| ~て**いただく** | 윗사람에게 받다 (정중) |
| ~て**くださる** | 윗사람이 내게 해주시다 |
| ~て**さしあげる** | 내가 윗사람에게 해드리다 (직접 쓰면 무례할 수도) |

### 4. 의뢰·은혜 표현

- 「友達に**手伝ってもらった**」 = 친구가 도와줬다 (내가 부탁해서)
- 「先生に**教えていただきました**」 = 선생님께 배웠습니다 (정중)
- 「友達が**手伝ってくれた**」 = 친구가 (자발적으로) 도와줬다

「もらう」vs「くれる」: 발화 시점에서 **부탁의 뉘앙스가 있으면 もらう**, **자발적 행위가 강조되면 くれる**.

### 5. 사역수수 (使役 + 授受)

특허·허가·요청을 정중하게 표현.

- 「~(さ)せていただく」: 「제가 ~해도 될까요」/「~하겠습니다」 (정중·겸양)
  - 例: 「明日**休ませていただきます**」 (내일 쉬겠습니다)
  - 例: 「ご説明**させていただきます**」 (설명드리겠습니다)
- 「~(さ)せてもらう」: 가까운 사이 (해도 돼요?의 겸양 표현)
  - 例: 「先に**帰らせてもらう**ね」
- 「~(さ)せてください」: 「~하게 해주세요」 (허가 요청)
  - 例: 「もう一度**やらせてください**」

### 6. 비교 정리

| 시점 | 표현 | 뉘앙스 |
|---|---|---|
| 윗사람이 내게 호의 | ~てくださる / ~ていただく | 「いただく」가 더 겸양 |
| 내가 윗사람에게 호의 | ~てさしあげる | 직설적이면 실례 — 우회 권장 |
| 허가 요청 | ~させていただきます | 비즈니스 정중 |
| 가벼운 요청 | ~させてください | 회화체 |

## JLPT N3 빈출 패턴

- 「先生に~ていただきました」: 정중한 은혜 표현
- 「~させていただきます」: 비즈니스 메일·발표 시작 표현
- 「友達に / が ~てくれた / もらった」: 시점 차이 묻기
- 「お父さんに~てもらった」: 누가 누구에게의 방향 묻기

## 예문

1. 友達がプレゼントを**くれました**。 — 친구가 선물을 줬다.
2. 先生に日本語を**教えていただきました**。 — 선생님께 일본어를 배웠다.
3. 妹に宿題を**手伝ってあげた**。 — 여동생의 숙제를 도와주었다.
4. 明日、お休みを**取らせていただきます**。 — 내일 휴가를 받겠습니다.

## 연습 문제

1. 田中さんは私に本を___。
   (A) あげました  (B) もらいました  (C) くれました  (D) やりました

2. 課長に書類を見て___。
   (A) いただきました  (B) くださいました  (C) さしあげました  (D) もらいまた

3. 先に帰___ですか。
   (A) させていただいてもいい
   (B) させてもらってもいい
   (C) させてあげてもいい
   (D) させてくれてもいい

**정답**

1. (C) くれました — 남→나, 일반 정중
2. (A) いただきました — 윗사람에게 받음 (정중)
3. (A) させていただいてもいい — 「제가 ~해도 될까요」 정중 표현
