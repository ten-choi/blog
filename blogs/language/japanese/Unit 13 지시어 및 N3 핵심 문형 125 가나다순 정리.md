## 개요

문법 후반부의 종합 정리. 지시어 こそあど의 응용을 다지고, N3 핵심 문형을 알파벳·발음순으로 한눈에 정리한다. 각 문형은 「의미 + 접속 + 예문 1」으로 압축.

## 핵심 문법

### 1. 지시어(こそあど) 응용

- **현장 지시**: 화자·청자 위치 기준 (Unit 02 참고)
- **문맥 지시**: 글·대화에서 이미 언급된 것 받음
  - 화자만 알고 청자는 모름 → **あ** 계열: 「あの店、本当に美味しかった」
  - 둘 다 알고 있음 → **そ** 계열: 「そのとき、私はまだ知らなかった」
- **추상·심리 지시**: 「**そういう**人」「**こういう**場合」

| 의미 | こ | そ | あ | ど |
|---|---|---|---|---|
| 그런 종류 | こんな | そんな | あんな | どんな |
| 그렇게 | こう | そう | ああ | どう |
| 그런 (の) | この | その | あの | どの |

### 2. N3 핵심 문형 — 의미·접속·예문

가나다(あいうえお)순. 각 항목 = 「**문형**: 의미 — 접속 — 예문」

#### あ행

- **~あいだ(に)**: ~동안(에) — 동사 사전형/명사+の — 「映画を見ているあいだに、電話があった」
- **~あまり**: ~한 나머지 — V사전형/N+の — 「驚いたあまり、声が出なかった」

#### い·う행

- **~一方(いっぽう)で**: 한편으로 — V사전형 — 「便利な一方で、危険もある」
- **~うえに**: 게다가 — V·N+の·い형용사·な형용사+な — 「安いうえに、量も多い」
- **~うちに**: ~동안에·~사이에 — V사전형/V-ない/N+の — 「若いうちに、たくさん勉強しなさい」

#### お·か행

- **~おかげで**: 덕분에 — V·N+の — 「先生のおかげで合格できた」
- **~がたい**: ~하기 어렵다 (감정·평가) — Vます — 「信じがたい話」
- **~かねる**: ~하기 어렵다 (정중) — Vます — 「お答えしかねます」
- **~がる**: ~해 한다 (3인칭 감정) — い형용사 어간 — 「子供は寒がる」
- **~かわりに**: 대신에 — V사전형 — 「行くかわりに電話する」

#### き·く·け행

- **~きる / きれない**: 다 ~하다 / 다 ~할 수 없다 — Vます — 「全部食べきった」「食べきれない」
- **~くせに**: ~인 주제에 — V·N+の·い·な+な — 「子供のくせに偉そうだ」

#### こ행

- **~ことになる**: ~하게 되다 (결정) — V사전형/V-ない — 「来月、出張することになった」
- **~ことにする**: ~하기로 하다 (의지) — V사전형/V-ない — 「明日から走ることにした」
- **~ことがある**: ~한 적이 있다 / 가끔 ~한다 — Vた / V사전형 — 「日本に行ったことがある」

#### さ·し·す행

- **~さえ**: ~조차 — N — 「子供さえ知っている」
- **~さえ~ば**: ~만 ~하면 — 다양 — 「お金さえあれば」
- **~しかない**: ~할 수밖에 없다 — V사전형 — 「待つしかない」
- **~ずに**: ~하지 않고 — V-ない (する→せず) — 「朝ごはんを食べずに出かけた」

#### た·ち·て·と행

- **~たことがある**: 「ことがある」와 동일
- **~たびに**: ~할 때마다 — V사전형/N+の — 「会うたびに笑顔(えがお)になる」
- **~たまま**: ~한 채로 — Vた — 「靴をはいたまま入った」
- **~ついでに**: ~하는 김에 — V사전형/Vた — 「買い物のついでに郵便局に寄った」
- **~ところ**: ~하려는 / ~한 직후 — V사전형/Vている/Vた — 「今、出るところだ」
- **~として**: ~로서 — N — 「医者として働く」
- **~とともに**: ~과 함께·동시에 — N/V사전형 — 「成長とともに考えも変わる」

#### な행

- **~ながら**: ~하면서 — Vます — 「歩きながら話す」
- **~なんか / なんて**: ~따위 — N — 「ゲームなんて、しない」
- **~にくい**: ~하기 어렵다 — Vます — 「読みにくい字」
- **~にしたがって**: ~함에 따라 — V사전형/N — 「時代にしたがって変わる」
- **~について**: ~에 대해 — N — 「環境について話す」
- **~によって**: ~에 의해, ~에 따라 — N — 「人によって意見が違う」

#### は·ふ·ほ행

- **~ば~ほど**: ~할수록 — V·い·な·N — 「読めば読むほど面白い」
- **~ばかり**: ~만, 갓 ~함 — N/Vた — 「遊んでばかりいる」「来たばかり」
- **~ばかりでなく**: ~뿐 아니라 — N/V사전형 — 「英語ばかりでなく中国語もできる」
- **~ほど**: ~할 정도 — V·N — 「泣くほどうれしかった」

#### ま·み행

- **~まま**: ~한 채 — Vた/N+の/형용사 — 「電気をつけたまま寝た」
- **~みたい**: ~같다 (구어) — V·N — 「子供みたい」

#### よ·ら·わ행

- **~ようとする**: ~하려고 하다 — V의지형 — 「立ち上がろうとした」
- **~ように**: ~하도록 — V사전형/V-ない — 「忘れないように書いておく」
- **~ようになる**: ~하게 되다 (능력·습관) — V사전형/Vられる — 「日本語が話せるようになった」
- **~らしい**: ~답다·~인 것 같다 — N/V·い·な — 「子供らしい」「彼は来ないらしい」
- **~わけがない / わけにはいかない**: ~할 리 없다 / ~할 수는 없다 — V·い·な — 「知らないわけがない」

## JLPT N3 빈출 패턴

- **빈도 1위급**: ~ようになる, ~ことにする, ~ことになる
- **공기 표현**: 「~するついでに / ~したことがある / ~ばかりだ」
- **혼동 주의**: 「ことになる(외부 결정)」 vs 「ことにする(내 결정)」
- **고급 어휘 연동**: 「~について / ~によって / ~とともに」는 독해 빈출

## 예문

1. 日本語が話せる**ようになった**。 — 일본어를 말할 수 있게 되었다.
2. 来月、引っ越し**することになった**。 — 다음 달, 이사하게 되었다.
3. 健康のために、毎朝走る**ことにした**。 — 건강을 위해 매일 아침 달리기로 했다.
4. 読めば読む**ほど**、その本が好きになった。 — 읽으면 읽을수록 그 책이 좋아졌다.

## 연습 문제

1. 田中さんの___で、無事に試験に合格できた。
   (A) おかげ  (B) せい  (C) ため  (D) ばかり

2. 毎日練習した結果、ピアノが弾ける___。
   (A) ことにした  (B) ことになった  (C) ようになった  (D) ようにした

3. このお菓子は子供___大人にも人気がある。
   (A) ばかり  (B) ばかりで  (C) ばかりでなく  (D) なんか

**정답**

1. (A) おかげ — 긍정적 원인 (덕분에)
2. (C) ようになった — 능력의 변화
3. (C) ばかりでなく — 「~뿐 아니라 ~도」
