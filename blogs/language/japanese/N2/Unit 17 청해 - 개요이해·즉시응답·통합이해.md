## 개요

N2 聴解(청해)의 후반 — ① **개요이해(概要理解)** = 글 전체의 주제·요지 파악, ② **즉시응답(即時応答)** = 짧은 발화에 즉답, ③ **통합이해(統合理解)** = 긴 대화·정보를 들으면서 선택지를 비교해 종합 판단. 셋 다 **선지(選択肢)를 미리 안 본 채로** 풀어야 하는 특성 때문에 사전 청취 훈련이 필수.

## 핵심 문법

### 1. 출제 구성

| 영역 | 문제 수 | 길이 | 핵심 |
|---|---|---|---|
| 概要理解 | 5문제 | 약 1분 모놀로그 | 「何の話か / 何が言いたいか」 |
| 即時応答 | 12문제 | 1~2 문장 | 가장 적절한 응답 선택 |
| 統合理解 | 4문제 | 약 2~3분 + 정보 비교 | 종합 판단 |

### 2. 개요이해 — 풀이 전략

1. **선지가 미리 안 보임** → 글을 다 들은 후 선지 4개 듣고 선택
2. 광고·뉴스·강의·연설·인터뷰가 주 소재
3. **첫 1~2 문장**에 화제, **마지막 1문장**에 요지가 압축됨
4. 「**~について話しています / これからお話しするのは~です**」 → 화제 단언 신호
5. 「**つまり / 要(よう)するに / ようするに / 結局(けっきょく)**」 → 결론 직전

### 3. 즉시응답 — 풀이 전략

1. **1문제당 약 10~15초** — 짧은 발화 후 3개 응답 중 자연스러운 것 선택
2. 함정 1: **발화에 나온 단어를 응답에 그대로 반복**한 선지는 거의 오답
3. 함정 2: 발화의 시제·주어를 잘못 받은 응답
4. **존경·겸양·정중**의 격식 일치 — 비즈니스 톤이면 응답도 정중하게
5. **의문문이 아닌 발화**에도 응답 필요 (감사·사과·정보 제공)

### 4. 즉시응답 — 발화 유형별 응답 패턴

| 발화 유형 | 자연스러운 응답 |
|---|---|
| 의뢰: 「~ていただけませんか」 | 「はい、かしこまりました」「分かりました」 |
| 사과: 「申し訳ありません」 | 「いえいえ、お気になさらず」 |
| 감사: 「ありがとうございます」 | 「いえ、こちらこそ」 |
| 권유: 「~はいかがですか」 | 「ぜひお願いします」「すみません、結構です」 |
| 정보 제공 | 「そうですか、ありがとうございます」 |
| 의외·놀라움 | 「えっ、本当(ほんとう)ですか」 |
| 거절 (정중) | 「申し訳ありませんが、ちょっと~」 |
| 양해: 「~でもよろしいですか」 | 「もちろんです」「はい、大丈夫です」 |

### 5. 통합이해 — 풀이 전략

- 가장 어려운 영역. 긴 정보 (라디오·강의·점원 설명)와 두 사람의 대화가 결합되어 출제.
- 「점원이 4종류 상품을 설명 → 남녀가 어떤 것을 살지 의논 → 정답 선택」
- **메모 필수** (시험지에 가능). 각 옵션의 키워드만 짧게 적기.
- 자주 묻는 유형:
  - 「男(おとこ)の人(ひと)はどれを選びますか」
  - 「2人はどれにしますか」
  - 「男の人と女の人で違うものを選ぶことになる場合、女の人は何を選びますか」

### 6. 통합이해 — 메모 요령

- 등장 옵션을 종이 위에 「①②③④」로 적고 핵심 속성 1~2개씩 표시
  - 예: ①安い·黒 / ②高い·カラー / ③軽い·小さい / ④定番
- 대화 중 「**~は~だから無理(むり)ね / それはちょっと~**」 — 제외 신호
- 「**~の方(ほう)がいいんじゃない**」 — 선호 신호
- 마지막 결정의 핵심 키워드를 옵션과 매칭

### 7. 통합이해 결정 단서 어휘

| 단서 | 의미 | 신호 |
|---|---|---|
| やっぱり / やはり | 역시 (번복·재확인) | 결정 번복 직전 |
| それなら / でしたら | 그렇다면·그럼 | 결정 도입 |
| かえって / むしろ | 오히려 (역접 결론) | 우선순위 뒤바뀜 |
| ~の方(ほう)がいい | ~쪽이 좋다 | 선호 신호 |
| ~は無理(むり)ね / ~はちょっと~ | ~은 무리·~은 좀 | 제외 신호 |
| ~ことにする / ~にする | ~로 하다 (의지 결정) | 최종 채택 |
| 決(き)めた / 決まり | 결정! | 명시적 결정 |
| 見送(みおく)る / 後回(あとまわ)し | 보류 | 거절·연기 |

### 8. 청해 어휘 — N2 단계 추가 표현

| 표현 | 의미 |
|---|---|
| ~に過(す)ぎない | ~에 불과하다 |
| ~とは限(かぎ)らない | ~라고는 단정할 수 없다 |
| ~わけにはいかない | ~할 수는 없다 |
| ~に違(ちが)いない | ~임에 틀림없다 |
| ~ものだ | ~인 법이다 |
| ~わけだ | ~인 셈이다 |
| ~ことか | ~인지! (감탄) |
| ~ところだ | ~참이다 / 막 ~한 |
| ~ばかりだ | ~할 뿐이다 |

## JLPT N2 빈출 패턴

- **개요이해**: 화제 단언 「**~について話します**」 → 핵심 키워드는 그 뒤
- **즉시응답**: 발화 단어 반복 = 함정. **상황에 맞는 자연스러운 응답** 선택
- **통합이해**: 「**~の方がいい·~は無理**」 가 선호·제외 신호
- 「**~とは限らない·~わけではない**」 부분 부정 함정 — 「100% 부정」 선지 오답
- 「**検討させていただきます**」 = 거의 거절

## 예문 — 개요이해 모놀로그 샘플

> 最近、若い人の中で「タイパ」という言葉(ことば)が流行(はや)っています。これは「タイムパフォーマンス」の略(りゃく)で、時間あたりの効果や満足感を意味します。たとえば、映画を1.5倍速で見たり、SNSの動画を短い時間で大量(たいりょう)に消費(しょうひ)したりすることが、その典型(てんけい)的(てき)な例です。
>
> しかし、本当(ほんとう)にすべての時間を効率(こうりつ)化(か)するのが幸せにつながるのでしょうか。私はそう思いません。なぜなら、人生(じんせい)には、ゆっくりと味(あじ)わうべき時間も多いからです。早送りした映画は、感動(かんどう)も半分(はんぶん)になってしまいます。
>
> 効率(こうりつ)を求(もと)めることと、心を満(み)たすことは、必ずしも一致(いっち)しません。それを、私たちはもう一度考え直すべきではないでしょうか。

**질문**: この人が一番言いたいことは何か。

- (A) タイパは若い人の流行語(りゅうこうご)である。
- (B) すべての時間を効率化すべきだ。
- (C) 効率化することが心を満たすとは限らない。
- (D) 映画は普通の速さで見るべきだ。

**정답**: (C) — 마지막 단락의 결론

## 예문 — 통합이해 모델 대화

**상황**: 카메라 가게 점원이 4종류의 디지털 카메라를 설명한다. 그 후 남자·여자가 어떤 모델을 살지 의논한다.

**점원의 설명**

> 本日(ほんじつ)はおすすめのデジタルカメラを4種類(しゅるい)ご紹介(しょうかい)します。
> ①「ライト100」 ─ 軽くてコンパクト、価格は2万円。初心者(しょしんしゃ)向け。
> ②「プロX」 ─ 高画質(こうがしつ)で連写(れんしゃ)に強い、価格は8万円。動きのある被写体(ひしゃたい)に最適(さいてき)。
> ③「ズームZ」 ─ 望遠(ぼうえん)が得意、価格は5万円。風景や鳥(とり)を撮(と)るのに向いています。
> ④「セルフィS」 ─ 自撮(じど)りに特化(とっか)、画面が回転(かいてん)します。価格は3万円。

**남녀 대화**

> 男: う~ん、僕は息子のサッカーの試合を撮りたいんだよね。動きが速くてさ。
> 女: それなら、連写が強いやつがいいんじゃない?
> 男: でも8万円はちょっと高いなあ…。
> 女: 私は旅行で景色を撮るのが好きだから、望遠が効くのがいいな。値段もそこそこだし。
> 男: ふ~ん。そうか、やっぱり予算オーバーは無理だから、僕は妥協(だきょう)して、軽くて持ち運びしやすいのにするよ。
> 女: 私はやっぱり望遠の方がいいわ。それにする。

**질문 1**: 男の人はどれを買いますか。
- (A) ①ライト100  (B) ②プロX  (C) ③ズームZ  (D) ④セルフィS

**질문 2**: 女の人はどれを買いますか。
- (A) ①ライト100  (B) ②プロX  (C) ③ズームZ  (D) ④セルフィS

**정답·풀이**

| 옵션 | 키워드 메모 | 평가 |
|---|---|---|
| ① ライト100 | 軽い·2만엔·초보 | 男이 「妥協して軽くて持ち運びしやすいのにする」 → 채택 |
| ② プロX | 8만엔·連写 | 男이 「高い」로 제외 |
| ③ ズームZ | 望遠·5만엔 | 女이 「望遠の方がいい·それにする」 → 채택 |
| ④ セルフィS | 자撮り·3만엔 | 둘 다 화제 안 함 |

- 질문 1: **(A) ①ライト100** — 「妥協して軽くて~にする」가 최종 결정
- 질문 2: **(C) ③ズームZ** — 「望遠の方がいい·それにする」 명시적 결정

## 즉시응답 — 연습

1. **発話**: 「お忙(いそが)しいところ、申し訳ありません」
   (A) お大事(だいじ)に。
   (B) いえ、大丈夫(だいじょうぶ)ですよ。何でしょうか。
   (C) もう少しお待ちください。

2. **発話**: 「コーヒー、もう一杯(いっぱい)いかがですか」
   (A) はい、いただきます。
   (B) いえ、お世話(せわ)になっております。
   (C) どうぞよろしくお願いします。

3. **発話**: 「明日の会議、3時に変更(へんこう)になりました」
   (A) かしこまりました。
   (B) お疲(つか)れさまでした。
   (C) いってらっしゃい。

**정답**

1. (B) — 사과에 대한 자연스러운 응답
2. (A) — 권유에 대한 수락
3. (A) — 정보 수령 시 「알겠습니다」

## 연습 문제

1. 청해 「**검토(けんとう)させていただきます**」 라는 응답이 들어왔다면, 가장 가까운 의미는?
   (A) 즉시 결정·실행
   (B) 거의 거절·보류
   (C) 추가 정보 요청
   (D) 결정권을 상대에게 넘김

2. 「~とは限らない」 의 의미는?
   (A) 항상 그러하다
   (B) ~라고는 단정할 수 없다 (부분 부정)
   (C) 절대 ~지 않다
   (D) 이미 ~했다

3. 통합이해 문제에서 가장 중요한 행동은?
   (A) 선지를 미리 본다
   (B) 등장 옵션의 핵심을 메모한다
   (C) 첫 단어만 듣고 판단한다
   (D) 발화 마지막 단어를 그대로 고른다

**정답**

1. (B) — 검토 표현은 보통 거절·보류의 정중 표현
2. (B) — 부분 부정
3. (B) — 옵션 메모가 통합이해의 핵심
