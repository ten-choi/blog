## 개요

N2의 敬語(경어)는 단순 변환에 그치지 않고, ① **특수형 동사 암기**, ② **존경·겸양·정중**(尊敬·謙譲·丁寧) 의 구분, ③ **이중경어 금지**, ④ **비즈니스 표준 표현**까지 묶어 출제된다. 문법 문제뿐 아니라 청해·독해에서 인물 관계 파악의 단서로 쓰인다.

## 핵심 문법

### 1. 3종 경어 구조

| 종류 | 대상 | 핵심 |
|---|---|---|
| 尊敬語(존경어) | 상대·제3자의 행위·상태 | 행위자를 높임 |
| 謙譲語(겸양어) | 자신의 행위 (상대에게 미침) | 자신을 낮춤 |
| 丁寧語(정중어) | 일반적 정중 표현 | ~です·~ます·~ございます |

### 2. 동사 특수형 — 외우는 게 가장 빠름

| 보통어 | 존경어(尊敬語) | 겸양어(謙譲語) |
|---|---|---|
| 行く / 来る | いらっしゃる / おいでになる | 参(まい)る / 伺(うかが)う |
| いる | いらっしゃる / おいでになる | おる |
| する | なさる | いたす |
| 言う | おっしゃる | 申(もう)す / 申し上げる |
| 食べる / 飲む | 召(め)し上がる | いただく |
| 見る | ご覧(らん)になる | 拝見(はいけん)する |
| 聞く | お聞きになる | 伺(うかが)う / 拝聴する |
| 知る | ご存(ぞん)じだ | 存じる / 存じ上げる |
| 会う | お会いになる | お目にかかる |
| 与える | くださる | 差(さ)し上げる |
| もらう | ─ | いただく / 頂戴(ちょうだい)する |
| くれる | くださる | ─ |
| 思う | お思いになる | 存じる |
| 借りる | ─ | 拝借(はいしゃく)する |
| 寝る | お休みになる | ─ |
| 死ぬ | お亡くなりになる | ─ |

### 3. 형식 변환 패턴 — 특수형이 없는 동사

| 형식 | 패턴 | 예 (書く) |
|---|---|---|
| 존경 | お+Vます+になる | お書きになる |
| 존경 | ~れる·~られる | 書かれる |
| 존경 (정중) | お+Vます+くださる | お書きください |
| 겸양 (型1) | お+Vます+する / いたす | お書きする·お書きいたします |
| 겸양 (型2) | お+Vます+申し上げる | お知らせ申し上げます |
| 한자어 | ご+한자어+になる(존)·する/いたす(겸) | ご説明になる·ご説明いたす |

### 4. 이중경어(二重敬語) 금지

- 한 동사에 경어 형식을 두 번 적용하지 말 것.
- **×** お読みになら**れる** (お~になる + ~れる)
- **×** お聞きになら**れる**
- **○** お読みになる / 読まれる (둘 중 하나만)
- 예외적으로 굳어진 「お召し上がりください·お見えになる」는 자연스럽다.

### 5. 「お+한자어 / ご+한자어」 — 「お/ご」 선택

- **お** + 훈독 / 음독 일부 (お電話·お時間·お部屋)
- **ご** + 음독 한자어 (ご家族·ご注文·ご案内)
- 외래어에는 거의 안 붙음 (「お コーヒー」 어색, 「コーヒー」 그대로)

### 6. 비즈니스 표준 표현

| 보통어 | 비즈니스 표현 |
|---|---|
| 今 | ただいま |
| すぐ | さっそく / 早急(さっきゅう)に |
| もう一度 | あらためて |
| 分かりました | かしこまりました / 承知(しょうち)いたしました |
| すみません | 申し訳(もうしわけ)ありません / 恐(おそ)れ入(い)ります |
| ちょっと | 少々(しょうしょう) |
| 誰 | どちら様(さま) |
| どう | いかが |
| そうです | さようでございます |

#### 자주 쓰는 정형구

- 「お世話(せわ)になっております」 — 신세 지고 있습니다
- 「お手数(てすう)ですが」 — 번거로우시겠지만
- 「恐れ入りますが」 — 죄송하지만 (의뢰 전)
- 「お忙(いそが)しいところ申し訳ございません」 — 바쁘신 와중에 죄송합니다
- 「ご確認(かくにん)のほど、よろしくお願いいたします」 — 확인 부탁드립니다

### 7. 자사·자기 가족 = 낮춤 표현

- 외부 사람에게 자사 사람을 말할 때:
  - 사장이 「申しました」 ○ (「おっしゃいました」 ×)
  - 부장이 「参ります」 ○
- 자기 가족을 외부에 말할 때:
  - 「父」「母」「兄」 (×お父さん·お母さん)

### 8. 사내·외부 호칭

| 자기 측 (謙称) | 상대 측 (尊称) |
|---|---|
| 弊社(へいしゃ) / 当社(とうしゃ) | 御社(おんしゃ, 회화) / 貴社(きしゃ, 서면) |
| わたくしども | 皆様(みなさま) / そちら様 |
| 父·母·兄·姉·祖父·祖母 | お父様·お母様·お兄様·お姉様·おじい様·おばあ様 |
| 主人·家内·妻·夫 | ご主人·奥様 |
| (사내) 部長の田中 / 田中 (이름만) | 田中部長·田中様·田中さん |

- 「お客様より弊社の田中に**お電話**がございました」 — 외부 손님이 자사 田中에게 → 田中에 호칭 없이
- 「御社のサービスについてお伺(うかが)いしたいのですが」 — 상대 회사는 「御社」

### 9. 「お+Vます+申し上げる·ご+한자어+申し上げる」 — 최고 격식 겸양

「~いたす」 보다 한 단계 위. 행사·서신·고객 응대에 사용.

- お知らせ**申し上げます** — 알려드립니다
- お祈り**申し上げます** — 기원드립니다
- お詫(わ)び**申し上げます** — 사죄드립니다
- ご報告(ほうこく)**申し上げます** — 보고드립니다
- ご案内**申し上げます** — 안내드립니다
- ご挨拶(あいさつ)**申し上げます** — 인사드립니다

## JLPT N2 빈출 패턴

- 「いらっしゃる·おっしゃる·召し上がる·ご覧になる」 등 **특수형 식별**
- 「**お/ご** + V+になる(존경)」 vs 「**お/ご** + V+する/いたす(겸양)」 식별
- 이중경어 함정: 「お読みになられる(×)·お聞きになられる(×)」
- 「**申す·いたす·参る·伺う·拝見する**」 겸양 5종 빈출
- 사내·가족에 대한 외부 발화의 격식

## 예문

1. 社長は3時に**いらっしゃる**予定です。 — 사장님은 3시에 오실 예정입니다.
2. お客様、お先に**召し上がって**ください。 — 손님, 먼저 드세요.
3. 私はあした、お宅(たく)に**伺います**。 — 저는 내일 댁으로 찾아 뵙겠습니다.
4. 先生が書いた論文を**拝見しました**。 — 선생님께서 쓰신 논문을 (제가) 보았습니다.
5. お忙しいところ、**恐れ入りますが**、少しお時間いただけますか。 — 바쁘신데 죄송하지만 시간을 좀 내주시겠습니까.

## 연습 문제

1. 「先生は今、何を___ますか」에서 「하고 계십니까」의 존경 표현은?
   (A) しています  (B) いたしています  (C) なさっています  (D) させています

2. 「明日、お客様のところに___」에 자연스러운 겸양 표현은?
   (A) 行きます  (B) いらっしゃいます  (C) 参ります  (D) おいでになります

3. 「お名前は何と___ますか」에 알맞은 존경 표현은?
   (A) 申し  (B) おっしゃい  (C) いたし  (D) 言い

4. 「お読み___」에서 자연스러운 존경 표현은?
   (A) になります  (B) になられます  (C) いたします  (D) されられます

**정답**

1. (C) なさっています — 「する」의 존경형
2. (C) 参ります — 「行く」의 겸양형
3. (B) おっしゃい — 「言う」의 존경형
4. (A) になります — 「お+Vます+になる」, B는 이중경어
