## 개요

N2 聴解(청해)의 1·2부 — **과제이해(課題理解)**와 **포인트이해(ポイント理解)**. N3와 구조는 같지만, ① 대화 길이가 1.5~2배, ② 어휘·표현이 격식·비즈니스 톤, ③ **번복·조건·예외**가 더 복잡하게 얽힌다. 「**누가 / 무엇을 / 언제 / 어디서 / 왜**」가 핵심.

## 핵심 문법

### 1. 출제 구성

| 영역 | 문제 수 | 길이 | 핵심 |
|---|---|---|---|
| 課題理解 | 5문제 | 짧은 대화 + 그림·문장 선택 | 「これからどうするか」 |
| ポイント理解 | 6문제 | 약 1분 대화 | 「なぜ / 何を / どこで」 |

### 2. 과제이해 풀이 전략

1. **질문이 먼저 나옴** → 무엇을 듣는지 명확히
2. 대화 중 **번복** 자주 등장:
   - 「**やっぱり** / **でも** / **じゃ** / **~に変更します**」 → 마지막 결정이 정답
   - 「**それなら**~にしましょう / **そうですね、では**~」
3. 등장인물이 둘 이상이면 **누구의 행동을 묻는지** 확인
4. 시간·순서 단어 주목:
   - 「**まず / 先(さき)に / 後(あと)で / 最後(さいご)に / 次に / その前に**」

### 3. 포인트이해 풀이 전략

1. 질문 후 **20초 정도 선택지 보는 시간** 제공 — 빠르게 키워드 점검
2. 이유 질문: 「**~から / ~ので / ~ため / だって / というのは**」 부분 집중
3. 부정문 함정:
   - 「~ないわけではない」 = 약한 긍정 (= ~인 면도 있다)
   - 「~ないこともない」 = 가능성은 있음
   - 「~とは言えない」 = 단언 못 함 = 약한 부정
4. 「~たいけど…」 = 결국 못 한다 / 「~たかったが、~した」 = 다른 행동을 함

### 4. 변경·번복 핵심 표현 (N2 단계)

- 「**やはり / やっぱり**~にします」 — 역시 ~로 할게요
- 「**じゃ**~に変更(へんこう)しましょう」 — 그럼 ~로 변경합시다
- 「**ちょっと考え直(なお)しまして**」 — 좀 다시 생각해 봤는데
- 「**それより**~の方(ほう)がいいかも」 — 그보다 ~가 더 좋을지도
- 「**やっぱりやめて**~」 — 역시 그만두고 ~
- 「**先(さき)に**~してから~します」 — 먼저 ~한 후에 ~할게요
- 「**今回(こんかい)は**見送(みおく)らせていただきます」 — 이번에는 보류하겠습니다 (= 안 함)
- 「**検討(けんとう)させていただきます**」 — 검토하겠습니다 (보통 보류·간접 거절)

### 5. 이유 묻기 빈출 표현

- 「**どうして / なぜ**~ですか / ~したのですか」
- 「**~わけ(理由)は何ですか**」
- 답변에 「**実は**~ / **というのは**~ / **というのも**~」 → 진짜 이유 단서
- 「**~ことになりまして**」 — 정중하게 결정·이유 알림

### 6. 비즈니스 빈출 표현 (청해 단골)

| 표현 | 의미 |
|---|---|
| 申し訳ございません | 죄송합니다 |
| お手数(てすう)ですが | 번거로우시겠지만 |
| 恐れ入ります | 죄송하지만·고맙습니다 |
| お見えになる | (손님이) 오시다 |
| 伺(うかが)う | 찾아 뵙다·여쭙다 |
| 承(うけたまわ)る | 받들어 듣다·접수하다 |
| かしこまりました | 알겠습니다 |
| 承知(しょうち)いたしました | 알겠습니다 (격식) |
| 後(のち)ほど | 나중에 |
| 早急(さっきゅう)に / 至急(しきゅう) | 시급히 |
| 一旦(いったん) | 일단 |
| 改(あらた)めて | 다시 한 번 |

### 7. 단서가 되는 문말 표현

- 「**~ことにする**」 = 화자의 결정 (의지)
- 「**~ことになる**」 = 외부 결정·결과 (의지와 무관)
- 「**~ようと思う**」 = 의지 (현재)
- 「**~つもりです**」 = 의지 (계획·예정)
- 「**~予定です**」 = 정해진 예정

## JLPT N2 빈출 패턴

- **번복 신호** 「やっぱり·それより·変更しまして」 = 정답은 그 뒤
- **간접 거절** 「検討させていただきます·見送らせていただきます」 = 거의 「안 한다」
- **이유 단서** 「実は」 뒤가 본심
- **시간 순서**: 「まず → 次に → 最後に」 의 「次に」 가 자주 문제의 답
- **부정 함정**: 「ないわけではない·ないこともない」 = 부정 아님

## 예문 — 청해 대화 샘플

**대화 (과제이해)**

> A(部下): 部長、来週の会議の資料、印刷しておきましょうか?
> B(部長): あ、お願いします。30部くらいで。
> A: かしこまりました。あ、それと、グラフの色は前回と同じでいいですか?
> B: いや、今回は白黒(しろくろ)で大丈夫(だいじょうぶ)です。コスト削減(さくげん)のために。
> A: 分かりました。じゃ、白黒で30部、印刷します。
> B: あ、すみません、やっぱり25部に変更してください。出席者(しゅっせきしゃ)が減(へ)ったので。

**질문**: 部下はこれから何枚印刷しますか。

- (A) カラーで30枚
- (B) 白黒で30枚
- (C) 白黒で25枚
- (D) カラーで25枚

**정답**: (C) — 마지막 번복 (「やっぱり25部に変更」) + 색은 백흑

## 연습 문제 — 청해 대화 (포인트이해)

**대화**

> A: あれ、田中さん、今日は車じゃないんですね。
> B: ええ、実は朝、車のエンジンが故障(こしょう)してしまって。
> A: 大変(たいへん)でしたね。修理(しゅうり)に出したんですか?
> B: ええ、でもすぐには直(なお)らないらしくて、1週間ほどかかると言われました。
> A: それは困りますね。じゃあ、しばらく電車で通勤(つうきん)ですか?
> B: ええ、でも電車だと遠回(とおまわ)りになるので、自転車(じてんしゃ)で来ることにしました。

1. 男性はなぜ今日車で来なかったのか。
   (A) 電車の方が早いから
   (B) 自転車で運動したかったから
   (C) 車が故障して修理に出しているから
   (D) 駐車場が満員(まんいん)だったから

2. 男性はしばらく何で通勤するか。
   (A) 車  (B) 電車  (C) 自転車  (D) バス

**정답**

1. (C) — 「実は~故障してしまって」 가 본심·이유
2. (C) — 마지막 결정 「自転車で来ることにしました」
