## 개요

N2 문법의 종합 정리. 시험 출제 빈도 상위 200개 문형을 **発音(あいうえお)순**으로 정렬. 각 문형은 「**의미 + 접속 + 예문 1**」으로 압축. 자주 충돌하는 문형은 비교 메모를 함께 둔다.

## 문형 200

### あ행

- **~あげく**: ~한 끝에 — Vた / Nの — 「悩んだあげく、辞めることにした」
- **~あまり**: ~한 나머지 — V사전형 / Vた / Nの — 「驚きのあまり言葉を失った」
- **~あっての**: ~이 있어야 비로소 — N — 「お客様あってのお店だ」

### い·う행

- **~いかんで / いかんによって**: ~여하에 따라 — N+の — 「成績いかんで進路が決まる」
- **~いかんでは / いかんによっては**: ~여하에 따라서는 — N+の — 「天候いかんによっては中止」
- **~いかんにかかわらず / いかんを問わず**: ~여하에 상관없이 — N+の — 「事情いかんにかかわらず実施する」
- **~一方(いっぽう)だ**: 점점 ~할 뿐 — V사전형 — 「物価は上がる一方だ」
- **~一方(いっぽう)で**: 한편으로는 — V사전형 / Nである — 「都会は便利な一方で、人間関係が希薄(きはく)だ」
- **~うえ(に)**: 게다가 — V·N·い·な+な — 「安いうえに美味しい」
- **~うえで**: ~한 후에 / ~함에 있어 — Vた / Nの — 「よく考えたうえで決める」
- **~うえは**: ~한 이상은 — Vた — 「約束したうえは守るべきだ」
- **~うちに**: ~동안에 — V사전형 / V-ない / N+の — 「明るいうちに帰ろう」

### お·か행

- **~おかげで**: 덕분에 — V·N — 「皆様のご協力のおかげで無事終わりました」
- **~恐(おそ)れがある**: ~할 우려가 있다 — V사전형 / Nの — 「事故の恐れがある」
- **~限(かぎ)り(は)**: ~하는 한 — V사전형 / Nである — 「体力がある限り働く」
- **~限りでは**: ~한 바로는 — V사전형 / Nの — 「私の知る限りでは」
- **~がたい**: ~하기 어렵다 (감정·평가) — Vます — 「彼の発言は理解しがたい」
- **~かたわら**: ~의 한편으로 — V사전형 / Nの — 「仕事のかたわら勉強する」
- **~がちだ**: ~하기 쉽다 — Vます / N — 「忘れがちだ」
- **~かと思うと**: ~한가 했더니 — Vた — 「雨が止んだかと思うと、また降り出した」
- **~か~ないかのうちに**: ~하자마자 — V사전형·Vない — 「ドアを開けるか開けないかのうちに猫が出た」
- **~かねる**: ~하기 어렵다 (정중·곤란) — Vます — 「個人情報につきましては申し上げかねます」
- **~かねない**: ~할지도 모른다 — Vます — 「事故になりかねない」
- **~かのようだ**: ~인 것처럼 — V·N — 「夢を見ているかのようだ」
- **~から見ると / 見れば**: ~로 보면 — N — 「子供から見れば不公平だ」
- **~からして**: ~부터가 — N — 「服装からして違う」
- **~からといって**: ~라고 해서 — V·N — 「高いからといっていいとは限らない」
- **~からには / からは**: ~한 이상은 — V사전형 / Vた — 「やるからには最後まで」

### き·く·け행

- **~気味(ぎみ)**: 약간 ~기 — N / Vます — 「疲れ気味」
- **~きり / ~たきり**: ~한 채로 — Vた — 「立ったきりだ」
- **~極(きわ)まりない**: ~하기 짝이 없다 — な형용사 어간 — 「失礼極まりない態度だ」
- **~きる / きれない**: 다 ~함 / 다 ~할 수 없음 — Vます — 「マラソンを走りきった」
- **~くせに**: ~인 주제에 — V·N·い·な+な — 「知らないくせに口を出すな」
- **~げ**: ~한 듯 — い형용사 어간 / な형용사 어간 — 「悲しげな顔」

### こ행

- **~ことから**: ~로부터 (이유) — V·い·な — 「事故の多いことから」
- **~ことだから**: ~이니까 — Nの — 「彼のことだから来るだろう」
- **~ことなく**: ~함이 없이 — V사전형 — 「休むことなく働く」
- **~ことに**: ~하게도 (감정) — Vた / い·な — 「驚いたことに」
- **~ところを**: ~한 와중에 — V사전형 / Nの — 「お忙しいところを恐れ入ります」

### さ·し·す행

- **~さえ**: ~조차 — N — 「専門家さえ予想できなかった」

### た·ち·つ행

- **~たて**: 갓 ~한 — Vます — 「焼きたてのパン」
- **~たとたん(に)**: ~한 순간 — Vた — 「ドアを開けたとたん、猫が逃げた」

- **~ついでに**: ~하는 김에 — V·Nの — 「散歩のついでにパンを買ってきた」
- **~っぱなし**: ~한 채로 — Vます — 「電気をつけっぱなし」
- **~つつ**: ~하면서 — Vます — 「考えつつ歩く」
- **~つつある**: ~하고 있다 (진행) — Vます — 「経済は回復しつつある」

### て·と행

- **~てからでないと**: ~한 후가 아니면 — Vて — 「確認してからでないと進められない」
- **~てしょうがない / てたまらない**: 너무 ~하다 — V·い·な+で — 「暑くてたまらない」
- **~とあって**: ~라서 (특수 상황) — N / V·い·な — 「連休とあって人が多い」

- **~というと / といえば**: ~라고 하면 — N — 「日本といえば富士山」
- **~というものの**: ~라고는 해도 — V·N·い·な — 「春とはいうものの寒い」
- **~どころか**: ~은커녕 — V·N·い·な — 「上達するどころか下手になった」
- **~どころではない**: ~할 처지가 아니다 — V·N — 「忙しくて遊ぶどころではない」
- **~とともに**: ~과 함께·~함에 따라 — N / V사전형 — 「経済成長とともに環境問題も深刻化した」

## JLPT N2 빈출 패턴

- **빈도 최상위**: 「~ものだ·~わけだ·~に違いない·~に決まっている·~に過ぎない·~に応じて·~に伴って」
- **혼동 주의**:
  - 「~**ばかりに**(나쁜 원인)」 vs 「~**おかげで**(좋은 원인)」 vs 「~**せいで**(부정 원인)」
  - 「~**わけがない**(개연 부정)」 vs 「~**わけではない**(부분 부정)」 vs 「~**わけにはいかない**(상황상 못함)」
  - 「~**かねる**(곤란해서 못함)」 vs 「~**かねない**(가능성 있음)」
  - 「~**ものの**(역접)」 vs 「~**ものだから**(변명)」
- **고급 어휘 연동**: 「~に基づいて·~に応じて·~をめぐって」 는 비즈니스·뉴스 독해 단골

## 예문

1. 経験を積む**につれて**自信がついてきた。 — 경험을 쌓음에 따라 자신이 붙었다.
2. 努力した**おかげで**合格できた。 — 노력한 덕분에 합격할 수 있었다.
3. 高い**からといって**、品質がいい**とは限らない**。 — 비싸다고 해서 품질이 좋다고 단정할 수 없다.
4. 彼は来ない**に違いない**。 — 그는 오지 않음에 틀림없다.
5. このアプリのおかげで、誰でも翻訳できる**ようになった**。 — 이 앱 덕분에 누구나 번역할 수 있게 됐다.
