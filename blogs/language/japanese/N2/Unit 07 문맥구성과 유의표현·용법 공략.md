## 개요

N2 文字·語彙(문자·어휘) 후반부의 문제는 ① **문맥규정**(괄호에 들어갈 단어 선택), ② **유의표현**(밑줄 친 단어를 같은 의미로 바꾸기), ③ **용법**(주어진 단어를 자연스럽게 사용한 문장 고르기). 단어 의미를 외우는 것에서 멈추지 않고 **호응 동사·문맥**까지 외워야 정답률이 오른다.

## 핵심 문법

### 1. 문맥규정 — 의미가 비슷해 헷갈리는 N2 빈출 어휘

| 단어 | 의미 | 호응 |
|---|---|---|
| 解決(かいけつ) | 해결 | 問題を解決する |
| 解消(かいしょう) | 해소 | ストレスを解消する |
| 修正(しゅうせい) | 수정 | 文書を修正する |
| 改善(かいぜん) | 개선 | 環境を改善する |
| 改革(かいかく) | 개혁 | 制度を改革する |
| 訂正(ていせい) | 정정 | 間違いを訂正する |
| 抑制(よくせい) | 억제 | 感情を抑制する |
| 制限(せいげん) | 제한 | 入場を制限する |
| 規制(きせい) | 규제 | 交通を規制する |
| 検査(けんさ) | 검사 | 健康を検査する |
| 検討(けんとう) | 검토 | 提案を検討する |
| 確認(かくにん) | 확인 | 内容を確認する |
| 観察(かんさつ) | 관찰 | 動物を観察する |

### 2. 유의어 — 의미는 비슷하나 뉘앙스가 다름

| 단어 | 단어 | 차이 |
|---|---|---|
| しばらく | ちょっと | しばらく(꽤 긴 시간) / ちょっと(짧음) |
| つまり | すなわち | つまり(구어·결론) / すなわち(문어·즉) |
| もちろん | むろん | もちろん(보통) / むろん(문어, 책에서) |
| たいてい | だいたい | たいてい(주로 빈도) / だいたい(주로 양·정도) |
| すぐ | 早速(さっそく) | すぐ(즉시) / 早速(서둘러·바로 행동) |
| とにかく | ともかく | とにかく(어쨌든 ~하다) / ともかく(~은 차치하고) |
| たびたび | しばしば | たびたび(자주 일상) / しばしば(자주 문어) |

### 3. 동사 유의어 — 사용 호응이 다름

| 단어 | 의미 | 차이 |
|---|---|---|
| 思う | 생각하다 | 일반적 |
| 考える | 사고하다 | 의식적·논리적 |
| 信じる | 믿다 | 신뢰·종교 |
| 確信する | 확신하다 | 강한 확정 |
| 伝える | 전하다 | 메시지·정보를 알림 |
| 伝わる | 전달되다 | 자동사 |
| 通じる | 통하다 | 의미·언어·길이 통함 |
| 加わる | 더해지다 | 자동사 (자연 추가) |
| 加える | 더하다 | 타동사 |
| 増える | 늘다 | 자동사 |
| 増やす | 늘리다 | 타동사 |

### 4. 부사·접속어 유의

- **やはり / やっぱり**: 역시 (예상대로) — 회화는 やっぱり
- **どうも / なんとなく**: 어쩐지 / 왠지 모르게 — 후자는 더 막연
- **しかも / さらに**: 게다가 / 더욱이 — しかも가 강함
- **ところで / さて**: (화제 전환) 그런데 / 자 — ところで는 갑작스러운 전환, さて는 시작·정리

### 5. 「~する」 동사가 되는 명사 — 의미 짝

| 명사+する | 의미 | 자/타 |
|---|---|---|
| 解決する | 해결하다 | 타 |
| 発生する | 발생하다 | 자 |
| 完成する | 완성하다 | 자/타 양용 |
| 成立する | 성립하다 | 자 |
| 普及する | 보급되다 | 자 |
| 進行する | 진행되다 | 자 |
| 進歩する | 진보하다 | 자 |
| 検討する | 검토하다 | 타 |
| 採用する | 채용하다 | 타 |
| 主張する | 주장하다 | 타 |

### 6. 용법 — 의미는 알지만 문장이 어색해지는 함정 단어

- **取り組む**: 「問題に取り組む」(문제에 임하다) — 추상적 과제에만
- **取り扱う**: 「商品を取り扱う」(상품을 취급하다) — 물건·업무에
- **当たる**: 「予想が当たる·宝くじに当たる」(맞다·당첨) — 사물 ×
- **及ぼす**: 「影響を及ぼす」(영향을 미치다) — 「影響をする」 ×
- **占める**: 「全体の半分を占める」(차지하다) — 비율
- **果たす**: 「役割を果たす·約束を果たす」(다하다·이행하다)
- **抱える**: 「問題を抱える·借金を抱える」(끌어안다·짊어지다)
- **務める / 努める / 勤める**:
  - 務める: 역할을 맡다 — 「司会を務める」
  - 努める: 노력하다 — 「改善に努める」
  - 勤める: 근무하다 — 「銀行に勤める」

## JLPT N2 빈출 패턴

- **~する 명사 + 호응 조사** 정확히: 「解決**に**取り組む·影響**を**及ぼす」
- 「**しばらく** vs **ちょっと**」 시간 폭 차이
- 「**たびたび** vs **しばしば**」 회화/문어
- 「**務める·努める·勤める**」 동음이의 한자 식별
- 「**取り組む·取り扱う**」 추상/구체 식별

## 예문

1. 環境問題に**取り組む**ことが大切だ。 — 환경 문제에 임하는 것이 중요하다.
2. このお店ではいろんな国の食品を**取り扱って**いる。 — 이 가게는 여러 나라의 식품을 취급한다.
3. 彼は会議で司会を**務める**。 — 그는 회의에서 사회를 맡는다.
4. 健康のために運動に**努めて**いる。 — 건강을 위해 운동에 힘쓰고 있다.
5. 父は銀行に**勤めて**いる。 — 아버지는 은행에 근무한다.

## 연습 문제

1. 「彼の発言が周囲に大きな影響を___」에 알맞은 것은?
   (A) した  (B) 及ぼした  (C) 行った  (D) 来た

2. 「私は会社に5年間___いる」에 들어갈 것은?
   (A) 努めて  (B) 勤めて  (C) 務めて  (D) 続けて

3. 「___その本は面白かった」에 가장 자연스러운 것은?
   (A) ちょっと  (B) しばらく  (C) もちろん  (D) かなり

4. 「彼の話は分かりにくい。___、結論があいまいだ」에서 가장 자연스러운 접속어는?
   (A) しかも  (B) ところで  (C) むしろ  (D) たとえば

**정답**

1. (B) 及ぼした — 「影響を及ぼす」 고정 호응
2. (B) 勤めて — 「~に勤める」 근무
3. (D) かなり — 정도 부사. 다른 선택지는 문맥 부적합
4. (A) しかも — 추가·강조의 게다가
