## 개요

JLPT N2 文字·語彙(문자·어휘) 파트의 출발점. N2 한자는 약 1,000자 수준이며, 단순 음·훈독 식별을 넘어 **복합어 안에서의 읽기 변화**, **熟字訓(숙자훈)**, 동음이의자(同音異義字) 구분을 묻는다. 문제 유형은 ① 한자→히라가나, ② 히라가나→한자, ③ 문맥상 가장 자연스러운 한자 선택.

## 핵심 문법

### 1. 음·훈독 심화 — 한자 단위가 아니라 「단어 단위」

- N3까지는 「漢字 1자 = 음독 1·훈독 1」 매핑이 통했지만, N2는 **단어 안에서 음이 변하는 현상**(連濁·連声·音便) 정복이 관건이다.
- 連濁(れんだく, 연탁): 뒷한자의 청음이 탁음으로 — 「花(はな) + 火(ひ) → 花火(はなび)」
- 連声(れんじょう): 「観(かん) + 音(おん) → 観音(かんのん)」 처럼 ん 뒤 음 변화
- 音便(おんびん): 「学校(がっこう, 가쿠+코우 → 갓코우)」 의 촉음화

### 2. 重箱読み·湯桶読み — 이중 음독

한 단어 안에서 음독+훈독 또는 훈독+음독이 섞이는 현상. N2 「자연스러운 한자 선택」에서 음·훈 혼동을 유도하는 단골 함정.

- **重箱(じゅうばこ)読み = 음독+훈독**

| 한자어 | 읽기 | 구성 |
|---|---|---|
| 重箱 | じゅう+ばこ | 음+훈 |
| 台所 | だい+どころ | 음+훈 |
| 本屋 | ほん+や | 음+훈 |
| 残高 | ざん+だか | 음+훈 |
| 番組 | ばん+ぐみ | 음+훈 |
| 王様 | おう+さま | 음+훈 |

- **湯桶(ゆとう)読み = 훈독+음독**

| 한자어 | 읽기 | 구성 |
|---|---|---|
| 湯桶 | ゆ+とう | 훈+음 |
| 朝晩 | あさ+ばん | 훈+음 |
| 場所 | ば+しょ | 훈+음 |
| 手本 | て+ほん | 훈+음 |
| 夕刊 | ゆう+かん | 훈+음 |
| 見本 | み+ほん | 훈+음 |

| 읽기 | 한자 | 의미 |

| たいしょう | 対象 / 対照 / 対称 | 대상 / 대조 / 대칭 |
| かんしん | 関心 / 感心 / 歓心 | 관심 / 감탄 / 호감 |

| 한자 | 音読み | 訓読み | 대표 단어 |
|---|---|---|---|
| 認 | にん | みとめる | 認識(にんしき), 認める(みとめる) |
| 与 | よ | あたえる | 与党(よとう), 影響を与える |
| 構 | こう | かまえる·かまう | 構成(こうせい), 構わない |
| 預 | よ | あずける·あずかる | 預金(よきん), 預ける(あずける) |
| 退 | たい | しりぞく·のく | 退院(たいいん), 退く(しりぞく) |

### 6. 한 글자에 음독이 둘 이상인 한자

| 한자 | 음독 1 | 음독 2 | 예 |
|---|---|---|---|
| 生 | せい | しょう | 学生(がくせい) / 一生(いっしょう) |
| 行 | こう | ぎょう / あん | 銀行(ぎんこう) / 行事(ぎょうじ) / 行脚(あんぎゃ) |
| 大 | だい | たい | 大学(だいがく) / 大切(たいせつ) |
| 下 | か | げ | 下水(げすい) / 地下(ちか) |
| 上 | じょう | しょう | 屋上(おくじょう) / 上人(しょうにん) |
| 人 | じん | にん | 日本人(にほんじん) / 人間(にんげん) |
| 月 | げつ | がつ | 月曜(げつよう) / 一月(いちがつ) |
| 道 | どう | とう | 道路(どうろ) / 神道(しんとう) |

- 시험 출제 시 「학교(学校)·학년(学年)」 등 같은 한자라도 후속 글자에 따라 음이 바뀌는 단어가 정답 함정으로 등장

### 7. 송가나(送り仮名) 함정

- 「行う」 vs 「行なう」 — 둘 다 가능, 시험은 「行う」
- 「表す / 現す / 著す」 — 의미별 한자 (나타내다 / 모습 보이다 / 책 펴내다)
- 「治める / 修める / 収める / 納める」 — 「おさめる」 4종

## JLPT N2 빈출 패턴

- 「**対象**」(대상) vs 「**対照**」(대조) — 문맥상 「~の対象になる/と対照的」 식별
- 「**保証**」(품질·신용) vs 「**保障**」(생활·안전) vs 「**補償**」(손해)
- 음·훈 혼합: 「**構造**(こうぞう, 음+음)」 vs 「**構える**(かまえる, 훈)」
- 숙자훈은 단독 출제 빈도 높음 — 「**田舎·眼鏡·為替**」 등 외워야 즉답

## 예문

1. 安全を**保障**(ほしょう)するための法律。 — 안전을 보장하는 법률.
2. 損害を**補償**(ほしょう)してもらった。 — 손해를 보상받았다.
3. 彼は強い**意志**(いし)の持ち主だ。 — 그는 강한 의지의 소유자다.
4. この問題は研究の**対象**(たいしょう)になっている。 — 이 문제는 연구 대상이 되고 있다.
5. お土産(**みやげ**)を買って帰った。 — 기념품을 사 가지고 돌아갔다.

## 연습 문제

1. 「将来の安全を**保障**する制度」의 保障 읽기는?
   (A) ほうしょう  (B) ほしょう  (C) ほしょ  (D) ぼしょう

2. 「かんしんがある」를 한자로 적절히 쓰면 (관심을 가지다)?
   (A) 関心  (B) 感心  (C) 歓心  (D) 寒心

3. 「為替」의 읽기는?
   (A) ためかえ  (B) ぎかえ  (C) かわせ  (D) いかえ

4. 「彼の意見に**対照**的だ」에서 対照의 의미로 가장 가까운 것은?
   (A) 대상  (B) 대조  (C) 대칭  (D) 대처

**정답**

1. (B) ほしょう — 「保障」 = 권리·생활을 지킴
2. (A) 関心 — 마음을 기울임. 「感心」은 감탄.
3. (C) かわせ — 숙자훈
4. (B) 대조 — 두 가지를 비교
