## 개요

N2 명사 학습의 핵심은 ① **추상명사·동작성 명사** 식별, ② **명사화 접미사**(さ·み·け·性·化)로 만들어진 파생어, ③ こそあど 대명사의 문맥 지시 응용. 단순 단어 암기 대신 **동사·형용사가 명사로 굳어지는 패턴**을 잡아야 작문·독해 점수가 오른다.

## 핵심 문법

### 1. N2 빈출 추상명사

| 명사 | 의미 | 같이 쓰는 동사 |
|---|---|---|
| 影響(えいきょう) | 영향 | 与える / 受ける |
| 印象(いんしょう) | 인상 | 残る / 受ける |
| 課題(かだい) | 과제 | 取り組む / 解決する |
| 傾向(けいこう) | 경향 | ~傾向がある |
| 現象(げんしょう) | 현상 | 起こる / 観察する |
| 効果(こうか) | 효과 | 効果がある / 期待する |
| 機会(きかい) | 기회 | 与える / 失う |
| 価値(かち) | 가치 | 認める / 高い |
| 態度(たいど) | 태도 | 取る / 示す |
| 立場(たちば) | 입장 | 立つ / 理解する |

### 2. 명사화 접미사 (動詞·形容詞 → 名詞)

| 접미사 | 의미 | 예 |
|---|---|---|
| ~さ | 객관적 정도 | 大きさ, 高さ, 重さ, 強さ |
| ~み | 주관적·내재된 성질 | 厚み, 深み, 楽しみ, 痛み |
| ~け | 어렴풋한 기색 | 寒け, 眠け, 人け |
| ~気(き) | 기운·기색 | 元気, 勇気(ゆうき), 平気 |
| ~性(せい) | ~성 | 可能性, 必要性, 重要性, 安全性 |
| ~化(か) | ~화 | 自動化, 国際化, 多様化, 高齢化 |
| ~度(ど) | ~도 | 満足度, 認知度, 難易度 |
| ~力(りょく) | ~력 | 想像力, 集中力, 体力, 影響力 |

### 3. 「~さ」 vs 「~み」 — 시험 단골

- **~さ**: 측정 가능·객관적 → 「重さは2キロ」「高さ100m」
- **~み**: 느낌·내재 → 「深みのある声」「人間味がある」
- 둘 다 가능한 경우 의미차:
  - 「楽しさ」 = 즐거운 정도 (객관 묘사)
  - 「楽しみ」 = 즐거움·기대감 (주관 감정·기대)
  - 「重さ」 = 무게(수치) vs 「重み」 = 무게감·진중함

### 4. 대명사 — こそあど 문맥 지시 응용

| 종류 | 현장 | 문맥 (양쪽 아는 것) | 문맥 (한쪽만 아는 것) |
|---|---|---|---|
| 사물 | これ·それ·あれ | それ | あれ |
| 장소 | ここ·そこ·あそこ | そこ | あそこ |
| 종류 | こんな·そんな·あんな | そんな | あんな |

- **そ계열 추상지시**: 글에서 막 언급된 내용 →「そういう問題」「そのような考え方」
- **あ계열 회상지시**: 화자만 알고 있는 과거의 일 →「あのときの感動」「あれはよかった」

### 5. 형식명사(形式名詞) — 본래 뜻이 흐려진 명사

| 형식명사 | 용법 |
|---|---|
| こと | 「~することがある」, 「~ということ」 |
| もの | 「~するものだ」(당위), 「~たものだ」(회상) |
| わけ | 「~わけがない」, 「~わけではない」 |
| ところ | 「~たところ」, 「~ところを見ると」 |
| はず | 「~はずだ」(예정·당연) |
| つもり | 「~つもりだ」(의도) |
| ため | 「~のために」(목적·원인) |

## JLPT N2 빈출 패턴

- 「重要**性**·必要**性**·可能**性**」 → ~성 시리즈 형태/의미 식별
- 「楽しい→楽しさ vs 楽しみ」 의미 차 구분
- 「**そんな**」 등 そ계열 추상지시 응용 — 문맥 일치 선택
- 형식명사 「もの·こと·わけ·はず」 의 차이 (특히 「~わけがない」 vs 「~はずがない」)

## 예문

1. 環境問題は社会に大きな**影響**(えいきょう)を与えている。 — 환경 문제는 사회에 큰 영향을 주고 있다.
2. この機械の**安全性**(あんぜんせい)が高く評価された。 — 이 기계의 안전성이 높게 평가됐다.
3. 彼の言葉には**深み**(ふかみ)がある。 — 그의 말에는 깊이가 있다.
4. **そのような**考え方は珍しい。 — 그러한 사고방식은 드물다.
5. 彼が嘘をつく**わけがない**。 — 그가 거짓말을 할 리가 없다.

## 연습 문제

1. 「もう少し___のある絵が描きたい」에 알맞은 것은?
   (A) 深さ  (B) 深み  (C) 深け  (D) 深性

2. 「彼の話には説得___がある」의 빈칸으로 알맞은 것은?
   (A) 化  (B) 度  (C) 力  (D) 風

3. 「___問題は子供にも分からない」에서 자연스러운 것은?
   (A) こういう  (B) こんな  (C) あんな  (D) そんな

4. 「明日は晴れる___」에서 의미가 「당연한 예상」인 것은?
   (A) はずだ  (B) わけだ  (C) ものだ  (D) ことだ

**정답**

1. (B) 深み — 작품 등 내재된 성질
2. (C) 力 — 설득력
3. (D) そんな — 화자·청자 다 알고 있음 (대화 맥락)
4. (A) はずだ — 합리적 예상·당연
