## 개요

N2 어휘 파트에서 가장 효율 높은 정복 대상이 **접두어(접두사)·접미어(접미사)**. 외워두면 모르는 단어도 의미 추정이 가능하다. 자주 출제되는 영역은 ① 부정·강조 접두어, ② 「~的·~性·~化·~風」 같은 추상화 접미어, ③ 인물·직업 접미어다.

## 핵심 문법

### 1. 부정 접두어 — 의미차 정복

| 접두어 | 의미 | 예 |
|---|---|---|
| 不(ふ) | 아닌·없음 | 不便·不安·不可能·不公平·不満 |
| 無(む / ぶ) | 없음 | 無理·無料·無事·無関心·無責任 |
| 非(ひ) | 그렇지 않음 (강한 부정) | 非常識·非公開·非公式·非科学的 |
| 未(み) | 아직 ~않음 | 未定·未満·未経験·未解決·未完成 |
| 反(はん) | 반대 | 反対·反応·反省·反論 |

#### 핵심 구별

- **不**: 「便利でない·安心でない」 → 「不」가 가장 보편적
- **無**: 「존재 자체가 없음」 → 無関心·無責任 (관심·책임 자체가 없음)
- **非**: 「상식·공식 등 정해진 기준에 어긋남」 → 非常識·非公式
- **未**: 「시간적·단계적으로 아직 안 됨」 → 未定·未完成

### 3. 경어·미화 접두어 「お·ご」

- **お** + 훈독계 명사: お茶·お金·お子さん·お店
- **ご** + 음독계 명사: ご家族·ご意見·ご注文·ご紹介
- 예외: お電話·お時間 (음독이지만 お), ご飯·ごゆっくり (특수)
- N2 시험에서는 「お/ご」 정확한 선택 문제 가능

| 접미어 | 의미 | 예 |

| ~性(せい) | ~성 | 必要性·可能性·重要性·安全性·個性 |
| ~化(か) | ~화 (변화) | 自動化·国際化·多様化·高齢化·活性化 |
| ~風(ふう) | ~풍·~식 | 和風·洋風·昔風·先生風 |
| ~式(しき) | ~식·~방식 | 日本式·新式·従来式·結婚式 |
| ~流(りゅう) | ~류·~식 | 自己流·我流·中国流 |
| ~観(かん) | ~관 | 人生観·世界観·価値観·歴史観 |
| ~論(ろん) | ~론 | 進化論·無神論·性悪論 |
| ~主義(しゅぎ) | ~주의 | 民主主義·個人主義·完璧主義 |

### 5. 상태·정도 접미어

| 접미어 | 의미 | 예 |
|---|---|---|
| ~気味(ぎみ) | ~기색·약한 ~경향 | 風邪気味·疲れ気味·遅れ気味 |
| ~がち | ~하기 쉬움·~경향 | 忘れがち·病気がち·遅れがち |
| ~だらけ | ~투성이 | 泥だらけ·血だらけ·間違いだらけ |
| ~まみれ | ~범벅 (액체·먼지) | 汗まみれ·泥まみれ·油まみれ |
| ~ずくめ | ~일색·~뿐 | 黒ずくめ·いいことずくめ |
| ~だけ | ~만·~뿐 | 一人だけ·これだけ |
| ~ぐらい / ~くらい | ~정도·~쯤 | 10分くらい·君ぐらい |

#### ~がち vs ~気味 (시험 단골)

- **~がち**: 비율·경향이 잦음 (객관·반복) — 「忘れがち·遅れがち」
- **~気味**: 살짝 그런 기색 (일시·약함) — 「疲れ気味·風邪気味」

#### ~だらけ vs ~まみれ

- **~だらけ**: 많이 묻어있음(추상 포함) — 「間違いだらけ·借金だらけ」
- **~まみれ**: 액체·가루 범벅 (구체적) — 「汗まみれ·泥まみれ」

### 6. 인물·직업 접미어

| 접미어 | 의미 | 예 |
|---|---|---|
| ~者(しゃ) | ~자 (전문) | 研究者·経営者·労働者·関係者 |
| ~家(か) | ~가 (예술·전문가) | 作家·画家·政治家·専門家 |
| ~人(じん) | ~인 (국적) | 日本人·外国人·個人 |
| ~員(いん) | ~원 (조직원) | 会社員·公務員·店員·会員 |
| ~師(し) | ~사 (기술자) | 医師·教師·美容師·料理師 |
| ~士(し) | ~사 (자격) | 弁護士·会計士·栄養士·運転士 |
| ~手(しゅ) | ~수 (직업·기술) | 選手·歌手·運転手 |

## JLPT N2 빈출 패턴

- 「不·無·非·未」 부정 접두어 정확히 선택 — 「**未**解決·**非**常識·**無**関心·**不**満」
- 「~性·~化·~的」 접미어 의미차 — 「経済**的**(상태) / 経済**化**(변화) / 経済**性**(성질)」
- 「~がち vs ~気味」 — 잘 출제
- 「~だらけ vs ~まみれ」 — 구체·추상 구분
- 「お/ご」 미화어 — 「ご家族·ご意見」「お茶·お金」

## 예문

1. この問題はまだ**未解決**だ。 — 이 문제는 아직 미해결이다.
2. 彼の言動は**非常識**だ。 — 그의 언동은 비상식적이다.
3. 子供は風邪を引き**がち**だ。 — 아이는 감기에 걸리기 쉽다.
4. 最近、疲れ**気味**でよく眠れない。 — 최근 피곤한 기색이라 잘 못 잔다.
5. 部屋がほこり**まみれ**だ。 — 방이 먼지투성이다.

## 연습 문제

1. 「この計画はまだ___定だ」에 알맞은 접두어는?
   (A) 不  (B) 無  (C) 未  (D) 非

2. 「子供は忘れ___なので、メモを書かせる」에 자연스러운 것은?
   (A) がち  (B) 気味  (C) だらけ  (D) ずくめ

3. 「彼の意見は具体___で分かりやすい」의 빈칸은?
   (A) 化  (B) 性  (C) 的  (D) 風

4. 「最近、運動不足___なので、ジムに通おうと思う」에 알맞은 것은?
   (A) がち  (B) 気味  (C) ずくめ  (D) まみれ

**정답**

1. (C) 未 — 「未定」 = 아직 정해지지 않음
2. (A) がち — 잦은 경향·습관
3. (C) 的 — 「具体的」 = 구체적
4. (B) 気味 — 약한 일시적 기색
