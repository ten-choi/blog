## 개요

N2 단계의 형용사는 활용·접속 자체는 N3와 같지만, ① **~がる·~がっている**(3인칭 감정), ② **~げ**(~한 듯), ③ **~み·~さ**(명사화), ④ **な형용사 어간의 부사형 응용**이 추가된다. 어휘 또한 추상도가 높아져 작문·독해에서 까다롭게 출제된다.

## 핵심 문법

### 1. 형용사 활용 정리표

| 형 | い형용사 | な형용사 | 명사 (비교용) |
|---|---|---|---|
| 기본형 | 高い | 静かだ | 学生だ |
| 부정 | 高くない | 静かではない | 学生ではない |
| 과거 | 高かった | 静かだった | 学生だった |
| 부정과거 | 高くなかった | 静かではなかった | 学生ではなかった |
| 명사 수식 | 高い+N | 静かな+N | 学生の+N |
| 부사형 | 高く | 静かに | ─ |
| て형 | 高くて | 静かで | 学生で |
| 가정 | 高ければ | 静かなら(ば) | 学生なら(ば) |
| 명사화 | 高さ/高み | 静かさ | ─ |

### 2. 「~がる·~がっている」 — 3인칭 감정·욕구

- 「私は寒い」 / 「彼は寒**がっている**」
- 1·2인칭은 자기 감정을 직접 말할 수 있지만, 3인칭은 추측·관찰을 통해 표현
- い형용사 어간 + がる: 寒**がる**·欲し**がる**·嫌**がる**
- な형용사 어간 + がる: 残念**がる**·嫌**がる**
- 「~たがる」: ~하고 싶어 한다 — 「子供が外に行きたがっている」

### 3. 「~げ」 — ~한 듯한 (문어·약간 옛스러움)

- い형용사 어간/な형용사 어간 + げ
- 寂し**げ**な顔 — 쓸쓸해 보이는 얼굴
- 自信あり**げ**な様子 — 자신 있어 보이는 모습
- 楽し**げ**に笑う — 즐거운 듯 웃다
- N2 독해에서 인물 묘사 자주

### 4. 명사화 — ~さ / ~み

- **~さ**: 객관·측정 가능 → 大き**さ**·広**さ**·静か**さ**
- **~み**: 주관·내재된 성질 → 厚**み**·深**み**·温か**み**·人間**み**·おもしろ**み**
- 둘 다 가능한 어간 의미 비교:
  - 「強さ」 = 강함의 정도 / 「強み」 = 강점
  - 「弱さ」 = 약함의 정도 / 「弱み」 = 약점·약한 부분
  - 「面白さ」 = 재미의 정도 / 「面白み」 = 묘미·맛

### 5. 형용사처럼 쓰는 명사·연어 표현

- **〜的(てき)**: 명사 → な형용사 — 「具体**的**な·経済**的**な·効果**的**な」
- **〜気味(ぎみ)**: 약한 ~기·~기색 — 「疲れ**気味**·風邪**気味**·遅れ**気味**」
- **〜らしい**: ~답다 / 추측 — 「学生**らしい**態度」「彼は来ない**らしい**」
- **〜っぽい**: ~한 느낌 — 「子供**っぽい**·安**っぽい**·忘れ**っぽい**」

### 6. い·な형용사 헷갈리는 어휘

| 단어 | 분류 | 의미 |
|---|---|---|
| 大きな | 연체사 (な형용사 아님) | 큰 |
| 小さな | 연체사 | 작은 |
| いろいろな | な형용사 | 여러 가지의 |
| 同じ | 연체사 | 같은 (×同じな) |
| 元気 | な형용사 | 건강한 |
| きれい | な형용사 (い처럼 보이지만 な) | 예쁜·깨끗한 |
| 嫌い | な형용사 | 싫어하는 |

## JLPT N2 빈출 패턴

- 「~**がる**」 3인칭 호응 — 「子供は新しいおもちゃを欲し**がっている**」
- 「~**げ**」 인물 묘사 — 「悲し**げ**·寂し**げ**·楽し**げ**」
- 「~**さ** vs ~**み**」 의미 차 — 「強さ vs 強み」
- 「同じ**な**·きれい**な**」 등 활용 함정
- 「**~的**+な+명사」 / 「**~的**+に+동사」 부사형 사용

## 예문

1. 弟は新しいゲームを欲しがっている。 — 동생은 새 게임을 갖고 싶어 한다.
2. 彼女は何か言いた**げ**な顔をしていた。 — 그녀는 무언가 말하고 싶은 듯한 표정이었다.
3. この絵には**温かみ**を感じる。 — 이 그림에는 따뜻함이 느껴진다.
4. 彼の**強み**はチームワークだ。 — 그의 강점은 팀워크다.
5. 最近、ちょっと疲れ**気味**だ。 — 최근 좀 피곤한 기색이다.

## 연습 문제

1. 「妹はそのお菓子を食べ___」에 적절한 것은?
   (A) たい  (B) たがっている  (C) たがる  (D) ほしい

2. 「彼の___は責任感の強さだ」에 자연스러운 것은?
   (A) 強さ  (B) 強み  (C) 強がる  (D) 強げ

3. 「最近、忙しくて寝不足___だ」에 들어갈 것은?
   (A) げ  (B) 気味  (C) っぽい  (D) らしい

4. 「これは経済___な問題だ」의 빈칸은?
   (A) 性  (B) 化  (C) 的  (D) 風

**정답**

1. (B) たがっている — 3인칭, 현재 상태
2. (B) 強み — 강점 (특질·장점)
3. (B) 気味 — 약한 경향·기색
4. (C) 的 — 「~的な+명사」 자주 쓰는 패턴
