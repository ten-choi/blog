## 개요

N2의 동사는 ① 자·타동사 짝(対応動詞), ② 의지·무의지 동사, ③ **복합동사**(~始める·~続ける·~切る…)가 핵심. 문법 문제뿐 아니라 어휘·독해 정답률을 좌우한다. 자·타동사 짝은 「의지·결과」 인지 「자연·상태」 인지의 의미 차이가 가장 중요하다.

## 핵심 문법

### 1. 자·타동사 짝(対応動詞) — N2 핵심 50쌍에서 30쌍 추림

| 자동사(自動詞) | 타동사(他動詞) | 의미 |
|---|---|---|

### 2. 자·타동사 사용의 결정적 차이

- **자동사 + ている** = 결과 상태 (의지 무관): 「窓が開いている」(창이 열려 있다)
- **타동사 + てある** = 누군가가 한 결과 (의지 함의): 「窓が開けてある」(창을 누가 열어 둔 상태)
- 「~が」 = 자동사, 「~を」 = 타동사 (기본)
- 의지 동사: 学ぶ·走る·書く — 「~ようと思う」 가능
- 무의지 동사: 困る·疲れる·分かる — 「~ようと思う」 부자연

### 3. 복합동사 — Vます + 보조동사

| 보조동사 | 의미 | 예 |
|---|---|---|
| ~始める | ~하기 시작하다 | 雨が降り**始めた** |
| ~出す | 갑자기 ~하기 시작 | 急に泣き**出した** |
| ~続ける | 계속 ~하다 | 走り**続ける** |
| ~終わる | 다 ~하다 | 読み**終わった** |
| ~切る | 끝까지 ~하다 | 食べ**切った** |
| ~切れない | 다 ~할 수 없다 | 食べ**切れない**ほど |
| ~かける | ~하려다·~을 향해 | 言い**かけて**やめた / 話し**かける** |
| ~直す | 다시 ~하다 | 書き**直す** |
| ~込む | 안으로 / 깊이 | 飛び**込む** / 考え**込む** |
| ~付ける | ~을 부착·습관화 | 押し**付ける** / 食べ**付けない** |
| ~過ぎる | 지나치게 ~하다 | 食べ**過ぎる** |
| ~忘れる | ~하기를 잊다 | 言い**忘れる** |

### 4. 의미 비슷한 복합동사 비교

- **~始める**(자연·중립 시작) vs **~出す**(갑작스러운 시작): 「降り始める」(예고된 비) / 「降り出す」(갑자기)
- **~切る**(끝까지·완전히) vs **~終わる**(작업 완료): 「使い切る」(다 써버리다) / 「使い終わる」(사용 끝)
- **~込む**(공간 안으로·깊게) — 「教え込む·しまい込む·考え込む」

## JLPT N2 빈출 패턴

- 「窓**が**開いて**いる**」(자동사+ている) vs 「窓**が**開け**てある**」(타동사+てある) 식별
- 「雨が降り**出した**」 = 갑자기 시작 → 「彼の話し中に急に~」 패턴
- 「彼は宿題を全部やり**切った**」 = 완수
- 자·타동사 짝의 형태 차이: 「~る/~す」 「~まる/~める」 「~がる/~げる」 패턴 인지

## 예문

1. ドアが**閉まって**いる。誰も触っていない。 — 문이 닫혀 있다. 아무도 손대지 않았다.
2. このドアは内側から**閉めてある**。 — 이 문은 안쪽에서 닫아 두었다.
3. 試合の途中で雨が**降り出した**。 — 시합 도중에 비가 갑자기 내리기 시작했다.
4. 借りた本を全部**読み終えた**。 — 빌린 책을 다 읽었다.
5. もう少しで諦め**かけた**が、頑張った。 — 거의 포기할 뻔했지만 노력했다.

## 연습 문제

1. 「電気が___まま、誰もいない」에 적절한 것은?
   (A) つけて  (B) ついた  (C) つけてある  (D) ついている

2. 「子供が急に___」에 가장 자연스러운 것은?
   (A) 泣き始めた  (B) 泣き出した  (C) 泣き終わった  (D) 泣き直した

3. 「忙しすぎて、仕事が___」에 가장 자연스러운 것은?
   (A) やり切れない  (B) やり始められない  (C) やり過ぎる  (D) やり込む

4. 「窓を___ください」에 알맞은 동사는?
   (A) 開く  (B) 開ける  (C) 開かる  (D) 開く

**정답**

1. (D) ついている — 자동사+ている, 단순 상태
2. (B) 泣き出した — 갑작스러운 시작 (「急に」와 호응)
3. (A) やり切れない — 다 처리할 수 없을 만큼 많음
4. (B) 開ける — 타동사 (~を)
