## 개요

N2 読解(독해) 1부, **단문(약 200자)**과 **중문(약 500자)**. N3와 비교해 문장 길이·어휘 난도·필자 의견의 함축도가 한 단계 올라간다. 단문은 1문제, 중문은 글당 3문제. 문제 유형은 ① 내용 일치, ② 필자의 의견·주장, ③ 밑줄의 의미·이유.

## 핵심 문법

### 1. 출제 구성

| 영역 | 글자수 | 글 수 | 문제 수 | 시간 배분 |
|---|---|---|---|---|
| 단문 | 200자 전후 | 5편 | 각 1문 | 글당 약 3분 |
| 중문 | 500자 전후 | 3편 | 각 3문 | 글당 약 6~7분 |

### 2. 단문 풀이 전략

1. **질문 먼저 본다** — 「筆者の考え」(필자의 생각) / 「~とあるが、何を意味するか」(밑줄 의미) / 「~の理由」(이유) 중 어느 유형인지
2. **첫 문장과 마지막 문장**에 결론·주장이 압축됨
3. **「のだ / だろう / と思う / べきだ」** 같은 문말 표현이 주관·객관·의무 판별 단서
4. 「**しかし·ところが·一方·むしろ**」 등장 시, 직후에 핵심 주장이 나옴

### 3. 중문 풀이 전략

1. 단락마다 **1문장 요약** (속독)
2. **접속사 신호**:
   - 「**つまり·要するに·結局**」 → 결론 직전
   - 「**しかし·ところが·一方**」 → 주장 방향 전환
   - 「**たとえば·具体的には**」 → 부차 정보 (정답 아닐 가능성↑)
3. **밑줄 친 부분의 이유**는 보통 **직전 문장 또는 직후 문장**
4. 선택지 4개 중 2개는 본문 표현 변형 — 본문 표현 그대로 vs 「~いう意味だ」 의 함정에 유의

### 4. N2 독해 핵심 키워드

| 종류 | 표현 | 의미 |
|---|---|---|
| 결론 | つまり / 要するに / 結局 / このように | 정답 직전 신호 |
| 역접 | しかし / ところが / だが / 一方 / もっとも | 주장 방향 전환 |
| 추가 | しかも / さらに / おまけに | 같은 방향의 강화 |
| 양보 | ~とはいえ / ~ものの / ~ながら | 일부 인정 |
| 인과 | だから / そのため / その結果 / ~ことから | 원인→결과 |
| 강조·진의 | むしろ / 実は / 確かに~が | 필자 진의 |
| 비판·우려 | ~とは限らない / ~わけではない / ~恐れがある | 부분 부정·우려 |

### 5. 「밑줄 친 ~とあるが」 유형 공략

- 「~とあるが、それは何を意味するか」 → 밑줄 **직전·직후 문맥**에서 동의어 찾기
- 「~とあるが、なぜか」 → 「**から·ので·ため·ことから**」 부분
- 「~とあるが、筆者はどう考えているか」 → 단락 마지막의 「**~と考えられる·~べきだ·~と思う**」

### 6. 자주 묻는 함정 어휘

- 「**確かに**~が、~」 = 「분명 ~지만, ~」 = 필자 진의는 「が」 뒤
- 「**~わけではない**」 = 부분 부정 (~인 것은 아님) — 본문에 있으면 「100% 부정」 선택지는 오답
- 「**~とは限らない**」 = ~라고 단정할 수 없음
- 「**~に過ぎない**」 = ~에 불과함

## 예문 — 단문 샘플

**다음 글을 읽고 질문에 답하시오.**

> 最近、若者の読書離(よみしょばな)れが指摘(してき)されている。しかし、私はその原因を一概(いちがい)にスマホのせいだとは思わない。確かに、スマホで動画を見たりゲームをしたりする時間は増えた。しかし、それと読書の時間が直接(ちょくせつ)結びつくとは限らない。むしろ、本を読まなくなったのは、自分の興味に合った本に出会う機会が減ったからではないだろうか。

**질문**: 筆者がこの文で最も言いたいことは何か。

- (A) 若者の読書離れはスマホのせいだ。
- (B) 若者の読書離れは興味に合う本に出会えないことが原因かもしれない。
- (C) 若者はスマホよりも本を読むべきだ。
- (D) 動画やゲームの時間を減らすべきだ。

**정답**: (B) — 마지막 문장 「むしろ~ではないだろうか」 가 필자의 진의

## 연습 문제

**다음 글을 읽고 답하시오.**

> 「働き方改革(かいかく)」という言葉(ことば)をよく耳にする。長時間労働(ろうどう)を減らし、休みを増やすことが目的だという。これは大切なことだろう。しかし、「働き方」を変えるだけで、すべての問題が解決(かいけつ)するわけではない。本当(ほんとう)に大切なのは、**働く意味**を一人ひとりが見つめ直すことだと私は考える。何のために働くのか。誰のために働くのか。その答えなくして、いくら制度を変えても、本質的(ほんしつてき)な改革(かいかく)にはならないだろう。

1. 筆者の主な主張は何か。
   (A) 長時間労働をすぐに減らすべきだ。
   (B) 働き方改革は不要だ。
   (C) 制度を変えるよりも、働く意味を考えることが大切だ。
   (D) 休みをもっと増やすべきだ。

2. 下線「働く意味」とはどういう意味か。
   (A) 仕事の給料の額
   (B) 自分が何のために誰のために働くのかという根本的な目的
   (C) 働く時間の長さ
   (D) 職場の人間関係

3. 「制度を変えても本質的な改革にはならないだろう」とあるが、その理由として正しいのは?
   (A) 制度を変えるのが難しいから
   (B) 働く意味が定まらないと制度だけ変えても根本解決にならないから
   (C) 長時間労働がなくならないから
   (D) 休みが少なすぎるから

**정답**

1. (C) — 마지막 문장의 결론
2. (B) — 「何のために·誰のために働くのか」 의 환언
3. (B) — 「働く意味」 의 부재가 본질적 한계
