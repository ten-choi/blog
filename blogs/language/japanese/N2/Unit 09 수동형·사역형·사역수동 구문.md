## 개요

N2의 수동·사역은 단순 형태 변환에서 끝나지 않고, ① **간접 수동(피해 수동)**, ② **사역의 두 가지 의미**(허가·강요), ③ **사역수동의 정중 표현**(~させていただく) 까지 확장된다. 비즈니스 일본어와도 직결된다.

## 핵심 문법

### 1. 수동형 형태 복습

| 동사 종류 | 변환 | 예 |
|---|---|---|
| 5단(1그룹) | ~う → ~われる | 言う→言われる, 書く→書かれる, 読む→読まれる |
| 1단(2그룹) | ~る → ~られる | 食べる→食べられる, 見る→見られる |
| 변격 | する→される / 来る→来られる | する→される, 来る→来られる |

### 2. 직접 수동 vs 간접 수동(피해 수동)

#### 직접 수동 (직접적인 행위의 대상)

- 「私は先生に**ほめられた**」 — 나는 선생님께 칭찬받았다
- 능동의 목적어가 수동의 주어가 됨

#### 간접 수동(피해 수동) — N2 시험 단골

목적어가 없는 자동사도 수동형이 가능. 화자가 **피해·곤란**을 입은 느낌을 강조.

- 「雨に**降られた**」 — 비를 맞았다 (자동사 「降る」의 수동)
- 「子供に**泣かれて**眠れなかった」 — 아이가 울어대서 잠을 못 잤다
- 「友達に**来られて**勉強できなかった」 — 친구가 와서 공부 못 했다
- 「上司に**辞められて**困った」 — 상사가 그만둬서 곤란했다

#### 「소유물의 수동」

- 「私は弟にケーキを**食べられた**」 — 동생이 (내) 케이크를 먹어버렸다
- 「彼は財布を**盗まれた**」 — 그는 지갑을 도둑맞았다
- 소유물이 피해 대상이 됨

### 3. 사역형 형태와 두 가지 의미

| 동사 종류 | 변환 | 예 |
|---|---|---|
| 5단 | ~う → ~わせる | 行く→行かせる, 書く→書かせる |
| 1단 | ~る → ~させる | 食べる→食べさせる |
| 변격 | する→させる / 来る→来させる | ─ |

#### 사역의 두 가지 의미

- **강제·지시 사역**: 「先生は学生に宿題をやらせた」 — 시켰다
- **허가 사역**: 「親は子供を遊ばせた」 — 허락했다
- 문맥으로 구별. 「~たい」와 함께 쓰이면 허가 의미 잦음.

#### 조사 차이

- 타동사: 「先生は学生**に**漢字を覚えさせた」 (に로 행위자 표시)
- 자동사: 「親は子供**を**外で遊ばせた」 (を로 행위자 표시)

### 4. 사역수동 — ~させられる

- 「(누가) ~하게 시킴을 당함」 → **억지로 ~함**
- 5단: 行く → 行かせる → 行か**される**(縮約形) / 行かせられる
- 1단: 食べる → 食べさせる → 食べさせられる
- する → させられる / 来る → 来させられる

#### 사용 예

- 「子供のとき、ピアノを**習わされた**」 — 어릴 때 피아노를 (억지로) 배웠다
- 「飲み会で歌を**歌わされた**」 — 회식에서 (억지로) 노래를 불렀다
- 「上司に残業**させられた**」 — 상사 때문에 야근하게 됐다

### 5. 정중한 사역 — ~させていただく

비즈니스 일본어의 핵심. 「내가 하는 행위에 대한 허가를 (상대로부터) 받는다」는 형식 → 매우 겸손하다.

- 「ご紹介**させていただきます**」 — 소개해 드리겠습니다 (자신을 낮춤)
- 「お休み**させていただきます**」 — 쉬겠습니다 (허락 받는 형식)
- 「ご説明**させていただいて**もよろしいでしょうか」 — 설명 드려도 될까요

#### 주의

- 자신의 행위에만 사용 (남의 행위에는 ×)
- 과용 시 어색함. 단순 「~いたします」가 충분한 경우도 많음.

### 6. 자발(自発)과 인접한 「불가항력」 표현

자발은 「자연스럽게 ~된다」, 「~ずにはいられない·~ないではいられない」 는 「~하지 않고는 못 배긴다」 — 둘 다 화자가 **의지로 통제할 수 없는** 행위·감정을 표현한다. 사역수동(외부 강제)와 의미상 대비된다.

| 표현 | 의미 | 접속 | 예 |
|---|---|---|---|
| ~ずにはいられない | ~지 않고는 못 배긴다 | V-ない (する→せず) | 「あの映画を見ると泣か**ずにはいられない**」 |
| ~ないではいられない | 同上 (회화체에 가까움) | V-ない | 「彼の話を聞くと笑わ**ないではいられない**」 |
| ~ずにはおかない | 반드시 ~하게 한다 (자동·필연) | V-ない | 「この事件は人々を驚かさ**ずにはおかない**」 |
| ~を禁(きん)じ得(え)ない | ~을 금할 수 없다 (격식) | N(감정명사) | 「悲しみ**を禁じ得ない**」 |

#### 사역수동과의 비교

- 「歌わ**された**」 (사역수동) = 외부 강제로 어쩔 수 없이
- 「歌わ**ずにはいられなかった**」 (불가항력) = 내적 충동으로 어쩔 수 없이
- 시험은 「~から」 이유절의 호응으로 식별: 「上司に言われて~된」 = 사역수동 / 「感動して~된」 = 불가항력

### 7. 「~される / ~られる」 형태 함정

- 「**~される**」: 수동 / 존경 (한자어+する의 존경어로도) — 「研究**される**」「ご出席**される**」
- 「**~られる**」: 수동 / 가능 / 존경 / 자발(自発) — 4가지 의미
- 「自発」 = 자연스럽게 ~되다 — 「故郷が思い**出される**」「将来が心配さ**れる**」

## JLPT N2 빈출 패턴

- **間接受身**(피해 수동) — 「雨に降られて·夜遅くに来られて」 의미 파악
- **사역의 두 의미** 식별 — 「やらせた」(시켰다 / 시켜 주었다) 문맥
- **사역수동** — 회식·근무 강요 묘사: 「飲まされた·歌わされた·残業させられた」
- 「**~させていただく**」 정중도·과용 식별
- 「~**られる**」 의 4가지 의미 (수동·가능·존경·자발) 식별

## 예문

1. 子供のとき、毎日ピアノを練習**させられた**。 — 어릴 때 매일 피아노 연습을 (억지로) 했다.
2. 夜中に隣の人に**騒がれて**眠れなかった。 — 한밤중에 옆 사람이 떠들어 잠을 못 잤다.
3. 父はいつも子供を自由に**遊ばせる**。 — 아버지는 늘 아이를 자유롭게 놀게 한다.
4. 来週、休暇を**取らせていただきます**。 — 다음 주 휴가를 받겠습니다.
5. このニュースを聞いて、昔のことが**思い出された**。 — 이 뉴스를 듣고 옛일이 자연스럽게 떠올랐다. (자발)

## 연습 문제

1. 「会議の途中で上司に___困った」에 자연스러운 것은?
   (A) 帰った  (B) 帰られて  (C) 帰らせて  (D) 帰らせられて

2. 「学生のとき、毎日漢字を___」에 알맞은 것은?
   (A) 書かれた  (B) 書かせた  (C) 書かされた  (D) 書いた

3. 「来週の月曜日、お休み___ます」에 가장 정중한 것은?
   (A) します  (B) させます  (C) させていただき  (D) されます

4. 「故郷の友人のことが思い___」에서 자발(自発)의 의미인 것은?
   (A) 出した  (B) 出される  (C) 出させた  (D) 出された

**정답**

1. (B) 帰られて — 간접 수동, 상사가 가버린 피해
2. (C) 書かされた — 사역수동, 강제로 ~함
3. (C) させていただき — 정중한 자기 행위 표현
4. (B) 出される — 자발, 자연스럽게 떠오름
