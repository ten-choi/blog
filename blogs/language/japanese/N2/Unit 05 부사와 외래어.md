## 개요

N2 부사는 단순 빈도부사를 넘어 **정도·시간·서법(서술 태도)·의태어·의성어** 까지 영역이 확장된다. 외래어 또한 비즈니스·경제·IT 어휘가 다수 출제되며, 의미·발음 두 가지를 모두 묻는다.

## 핵심 문법

### 1. 부사 5분류

| 분류 | 역할 | 대표 부사 |
|---|---|---|
| 정도 | 형용사·동사의 정도 강화 | とても·非常に·かなり·ずいぶん·相当(そうとう) |
| 빈도 | 동작의 횟수 | いつも·たまに·しばしば·度々(たびたび)·めったに |
| 시간 | 시간 흐름 | すぐ·しばらく·ずっと·やがて·とっくに |
| 서법 | 화자의 태도 | きっと·たぶん·おそらく·まさか·もしかして |
| 호응 | 특정 표현과 짝 | 全然(~ない)·決して(~ない)·ぜひ(~たい/~てください) |

### 2. 호응 부사 — 필수 짝궁

| 부사 | 짝꿍 표현 | 의미 |
|---|---|---|
| 全然(ぜんぜん) | ~ない | 전혀 ~않다 |
| 決して(けっして) | ~ない | 결코 ~않다 |
| めったに | ~ない | 좀처럼 ~않다 |
| 二度と(にどと) | ~ない | 두 번 다시 ~않다 |
| まったく | ~ない | 전혀 ~않다 |
| ちっとも | ~ない | 조금도 ~않다 |
| たぶん | ~だろう | 아마 ~일 것이다 |
| きっと | ~に違いない | 분명 ~할 것이다 |
| ぜひ | ~たい/~ください | 꼭 ~하고 싶다/해 주세요 |
| まるで | ~ようだ/~みたい | 마치 ~같다 |
| まさか | ~ないだろう | 설마 ~지는 않을 것이다 |
| もし | ~ば/~たら/~なら | 만일 ~라면 |

### 4. 의성어(擬音語) — 청해 빈출

자연·동물·신체 등에서 나는 소리를 흉내 낸 어휘. 청해 단원에서 「~の音(おと)が**ガタガタ**する」 같이 등장.

| 의성어 | 의미·용례 |
|---|---|
| ワンワン | 개 짖는 소리 — 「犬がワンワン鳴く」 |
| ニャー / ニャーニャー | 고양이 울음 |
| コケコッコー | 닭 우는 소리 |
| ザーザー | 비가 거세게 — 「雨がザーザー降る」 |
| シトシト | 비가 조용히 — 「シトシト降る雨」 |
| ガラガラ | 가게가 텅 빔·바퀴 굴러가는 소리 — 「店内はガラガラだった」 |
| カラカラ | 목 마름·바싹 마른 소리 — 「のどがカラカラだ」 |
| ガタガタ | 흔들림·낡음 — 「椅子がガタガタする」 |
| バタバタ | 분주함·발 구르는 소리 — 「朝からバタバタしている」 |
| ペコペコ | 배가 고픔·굽실거림 — 「お腹がペコペコだ」 |
| ドキドキ | 심장 두근거림 — 「胸がドキドキする」 |
| ハラハラ | 조마조마함 — 「ハラハラする試合」 |
| ピカピカ | 번쩍번쩍 빛남 — 「ピカピカに磨く」 |
| ガチャン | 깨지는 소리 — 「ガラスがガチャンと割れた」 |

### 5. 「わざわざ」 vs 「わざと」 — N2 단골

- **わざわざ**: 일부러(친절·수고) — 「わざわざ来てくださってありがとう」
- **わざと**: 일부러(고의·악의) — 「わざと負けた」「わざとぶつかった」

### 6. N2 빈출 외래어

| 외래어 | 일본어 | 분야 |
|---|---|---|
| データ | 자료 | IT·일반 |
| イメージ | 인상·이미지 | 일반 |
| サービス | 봉사·서비스 | 비즈니스 |
| プラン | 계획 | 비즈니스 |
| アイデア / アイディア | 발상 | 일반 |
| トレーニング | 훈련 | 스포츠·업무 |
| プレッシャー | 압박감 | 심리 |
| ストレス | 스트레스 | 심리·건강 |
| クレーム | 항의·민원 | 비즈니스 (※「칭찬」 아님) |
| インタビュー | 인터뷰 | 미디어 |
| アンケート | 설문 | 비즈니스 |
| キャンセル | 취소 | 비즈니스 |
| トラブル | 문제·말썽 | 일반 |
| バランス | 균형 | 일반 |
| ペース | 페이스·속도 | 일반 |

### 7. 영어 의미와 다른 「가짜 친구」 외래어

- **クレーム** ≠ claim 청구. **「항의·불만」**의 의미가 강함.
- **アルバイト** = part-time job (독일어 Arbeit 유래)
- **マンション** = 아파트(중층 이상). 영어의 mansion(저택) 아님.
- **サラリーマン** = 회사원 (일본식 영어)
- **コンセント** = 콘센트·전기 플러그 (영어 concent 아님; 영어로는 outlet)

## JLPT N2 빈출 패턴

- 「全然·決して·めったに」 + **ない** 호응
- 「**まさか**」+**「だろう/ないだろう」** — 강한 의외성
- 「わざわざ」(친절·수고) vs 「わざと」(고의)
- 「**やっぱり**」 = 「やはり」 회화체. 추측·예상이 맞았을 때.
- 외래어 의미 함정: 「**クレーム**」(불만·항의), 「**マンション**」(아파트)

## 예문

1. **わざわざ**遠くから来てくれてありがとう。 — 일부러 멀리서 와줘서 고마워.
2. 子供が**わざと**コップを落とした。 — 아이가 고의로 컵을 떨어뜨렸다.
3. その話を聞いて、**まさか**と思った。 — 그 이야기를 듣고 설마 했다.
4. 仕事のストレスでぐっすり眠れない。 — 일 스트레스로 푹 잘 수 없다.
5. お客様から**クレーム**が入った。 — 고객에게서 항의가 들어왔다.

## 연습 문제

1. 「___遠くから来てくださってありがとうございます」에 알맞은 것은?
   (A) わざと  (B) わざわざ  (C) たまたま  (D) すっかり

2. 「彼が嘘をつくなんて、___信じられない」에 자연스러운 것은?
   (A) きっと  (B) ぜひ  (C) まったく  (D) たぶん

3. 「___昨日のことのように覚えている」에 들어갈 것은?
   (A) まるで  (B) まさか  (C) ぜひ  (D) けっして

4. 「お店に___が入ったので対応が必要だ」의 빈칸은?
   (A) インタビュー  (B) クレーム  (C) アイデア  (D) サービス

**정답**

1. (B) わざわざ — 호의의 수고
2. (C) まったく — 「~ない」 호응
3. (A) まるで — 「~ようだ」와 호응
4. (B) クレーム — 고객 항의
