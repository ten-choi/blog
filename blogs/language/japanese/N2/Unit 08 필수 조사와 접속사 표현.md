## 개요

N2 조사·접속의 핵심은 **복합조사**(~について·~に対して·~に関して·~をめぐって 등) 사용. 의미가 비슷한 표현이 줄지어 있어 시험에서 자주 충돌한다. 각 표현의 ① **접속**(N·V형), ② **시점**(대상·관점·계기), ③ **자연스러운 호응**까지 묶어 외운다.

## 핵심 문법

### 1. 「대상·관계」 계열 복합조사

| 표현 | 의미 | 접속 | 예 |
|---|---|---|---|
| Nに対して | ~에 대해(서) (반응·태도) | N | 客に対して丁寧に話す |
| Nについて | ~에 대해 (서술의 주제) | N | 環境について話す |
| Nに関して | ~에 관해 | N | 事件に関して報道する |
| Nをめぐって | ~을 둘러싸고 | N | 法律をめぐって議論する |
| Nに応(おう)じて | ~에 따라·맞춰 | N | 状況に応じて対応する |
| Nに従(したが)って | ~함에 따라·~을 따라 | N/V사전형 | 指示に従って動く |
| Nに基(もと)づいて | ~을 근거로·기반으로 | N | データに基づいて判断する |
| Nを通(つう)じて | ~을 통해 | N | 友人を通じて知り合う |
| Nを通(とお)して | ~을 통해·내내 | N | 一年を通して暖かい |

#### 미세 차이 정리

- **に対して**: 대상에 대한 「반응·태도」 강조. 「子供に対して優しい」(아이에게 다정함)
- **について**: 화제·정보의 「내용」을 가리킴. 「日本文化について書く」
- **に関して**: 「について」의 정중·격식 표현. 비즈니스 문서에 자주.
- **をめぐって**: 화제 「주변에 의견·논쟁이 모여 있음」 — 논쟁·갈등 호응이 잦음.

### 2. 「관점·시점」 계열

| 표현 | 의미 | 예 |
|---|---|---|
| Nから見ると / 見れば / 見て | ~의 입장에서 보면 | 子供から見れば不公平だ |
| Nにとって | ~에게 있어 | 私にとって家族が大切だ |
| Nにすれば / すると | ~의 입장에서는 | 親にすれば子供が心配だ |
| Nとしては | ~로서는 (입장·역할) | 担当者としては困る |
| Nのもとで | ~ 아래에서·~ 밑에서 | 先生のもとで研究する |

#### 「にとって」 vs 「として」

- **にとって**: 평가·관점의 주체 — 「私**にとって**英語は難しい」(평가)
- **として**: 자격·역할 — 「先生**として**指導する」(자격)

### 3. 「때·시점」 계열

| 표현 | 의미 | 접속 |
|---|---|---|
| Vたとたん(に) | ~한 순간 | Vた |
| Vかと思うと | ~한가 했더니 (의외) | Vた |
| Vか~ないかのうちに | ~하자마자 | V사전형·Vない |
| Vたばかり | 막 ~함 | Vた |
| Vている最中(さいちゅう) | 한창 ~중 | V-て+いる |
| Vている間に | ~하는 동안에 | V-て+いる |
| ~とともに | ~과 함께·~함에 따라 | N / V사전형 |
| ~につれて | ~함에 따라 | V사전형 / N |
| ~に伴(ともな)って | ~에 따라·동반하여 | V사전형 / N |

#### 「につれて·に伴って·とともに·に従って」

- **につれて**: 점진적 변화 (자연·시간) — 「年をとる**につれて**体力が衰える」
- **に伴って**: 변화에 부수해 동시에 — 「人口増加**に伴って**問題も増える」
- **とともに**: 함께·동시에 (병행) — 「経済発展**とともに**環境問題が出てきた」
- **に従って**: ~을 따라(지시·규칙·자연) — 「ルール**に従って**行動する」

### 4. 「범위·기준」 계열

| 표현 | 의미 | 예 |
|---|---|---|
| Nを問わず | ~을 불문하고 | 国籍を問わず参加できる |
| Nにかかわらず | ~에 상관없이 | 天気にかかわらず開催 |
| Nはもとより | ~은 물론 | 日本人はもとより外国人にも人気 |
| Nに限(かぎ)らず | ~에 한정되지 않고 | 子供に限らず大人も楽しめる |
| ~に限る | ~이 최고다 | 夏はビールに限る |
| ~に限って | ~만은·~만큼은 | うちの子に限って嘘をつかない |

### 5. 「논리·결과」 접속

| 접속어 | 의미 | 용법 |
|---|---|---|
| そのため | 그래서 (인과) | 문어 |
| そこで | 그래서 (행동의 계기) | 「困った。そこで~した」 |
| それで | 그래서 (자연 결과) | 회화 |
| すると | 그러자 (의외 결과) | 「ボタンを押した。すると~」 |
| したがって | 따라서 (논리적 결론) | 격식 |
| ところが | 그런데 (의외·역접) | 「~と思った。ところが~だった」 |
| もっとも | 다만 (수정·단서) | 「賛成だ。もっとも~は除く」 |
| ただし | 단 (예외 조건) | 격식 문서 |

## JLPT N2 빈출 패턴

- 「~**について** vs ~**に対して**」 — 주제 / 반응 식별
- 「~**につれて** vs ~**に伴って**」 — 점진 / 동반 식별
- 「~**を問わず** vs ~**にかかわらず**」 — 의미 거의 같음, 호응 단어로 식별
- 「~**にとって** vs ~**として**」 — 평가 / 자격 구분
- 접속어 「**そこで·それで·すると·したがって**」 의 미세 차이

## 예문

1. このテーマ**について**もう少し詳しく説明します。 — 이 주제에 대해 좀 더 자세히 설명하겠습니다.
2. 先生は学生**に対して**いつも親切だ。 — 선생님은 학생들에게 늘 친절하다.
3. 経験を積む**につれて**、自信がついてきた。 — 경험을 쌓음에 따라 자신이 붙었다.
4. 性別**を問わず**、誰でも応募できる。 — 성별을 불문하고 누구나 응모할 수 있다.
5. 親**にとって**、子供の健康が一番だ。 — 부모에게 있어 자식의 건강이 최고다.

## 연습 문제

1. 「子供___厳しすぎる教育はよくない」에 알맞은 것은?
   (A) について  (B) に対して  (C) として  (D) によって

2. 「都市化___交通問題が悪化している」에 자연스러운 것은?
   (A) を問わず  (B) に伴って  (C) にとって  (D) として

3. 「年齢___誰でも参加できます」의 빈칸은?
   (A) を問わず  (B) によって  (C) について  (D) に応じて

4. 「人前で話すことは私___苦手だ」에 적절한 것은?
   (A) として  (B) について  (C) にとって  (D) によって

**정답**

1. (B) に対して — 「~에게 ~한 태도」 의미
2. (B) に伴って — 도시화에 따라 동시에 변화
3. (A) を問わず — ~를 불문하고
4. (C) にとって — 평가의 주체
