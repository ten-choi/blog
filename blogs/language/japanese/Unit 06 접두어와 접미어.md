## 개요

접두어(接頭語)·접미어(接尾語)는 N3 어휘 영역의 단골. 의미 패턴을 알면 모르는 단어도 추론 가능하다. 「不~」, 「~的」, 「~中」 같은 빈출 패턴을 패키지로 익힌다.

## 핵심 문법

### 1. 부정·강조 접두어

| 접두어 | 의미 | 예 |
|---|---|---|
| 不(ふ) | 부정 | 不便(ふべん), 不安, 不可能, 不満 |
| 無(む) | 없음 | 無料(むりょう), 無理, 無視, 無関心 |
| 非(ひ) | 비~ | 非常(ひじょう), 非公開, 非常識 |
| 未(み) | 아직 안 됨 | 未来(みらい), 未定, 未経験 |
| 反(はん) | 반대 | 反対, 反応, 反省 |

### 2. 정도·평가 접두어

| 접두어 | 의미 | 예 |
|---|---|---|
| 大(だい) | 큰 | 大成功, 大事(だいじ), 大好き |
| 高(こう) | 높은 | 高品質, 高水準(こうすいじゅん), 高価 |
| 低(てい) | 낮은 | 低価格, 低気圧, 低音 |
| 新(しん) | 새로운 | 新商品, 新人, 新作 |
| 旧(きゅう) | 옛, 구식 | 旧友, 旧式, 旧校舎 |
| 元(もと) | 원래의, 전(前) | 元社長, 元気, 元の場所 |

### 3. 사람·직책 접미어

| 접미어 | 의미 | 예 |
|---|---|---|
| ~者(しゃ) | ~하는 사람 | 担当者, 経験者, 関係者 |
| ~員(いん) | ~원, 직원 | 会社員, 駅員(えきいん), 店員 |
| ~家(か) | ~가, 전문가 | 専門家(せんもんか), 作家, 政治家 |
| ~師(し) | ~사 (기술자) | 医師, 教師, 美容師 |
| ~人(じん) | ~인 (사람) | 日本人, 外国人, 社会人 |
| ~様(さま) | ~님 (높임) | 田中様, お客様 |
| ~さん | ~씨 | 山田さん, 田中さん |
| ~ちゃん | (애칭) | 太郎ちゃん |

### 4. 상태·범주 접미어

| 접미어 | 의미 | 예 |
|---|---|---|
| ~的(てき) | ~적 (な형용사化) | 一般的, 具体的(ぐたいてき), 国際的 |
| ~性(せい) | ~성 | 可能性, 安全性, 重要性 |
| ~化(か) | ~화 (변화) | 国際化, デジタル化, 高齢化 |
| ~中(ちゅう) | ~중 | 仕事中, 食事中, 工事中(こうじちゅう) |
| ~式(しき) | ~식 | 日本式, 自動式, 卒業式 |
| ~風(ふう) | ~풍 | 和風, 洋風, 韓国風 |

### 5. 시간·횟수 접미어

- ~間(かん): 時間, 二日間, 週間 — 기간
- ~目(め): 二回目, 三日目, 一度目 — 순서
- ~ごと: 一日ごと, 二人ごと — ~마다
- ~ぶり: 三年ぶり, 久しぶり — ~만에

## JLPT N3 빈출 패턴

- **不**+な형용사 어간: 不便, 不安, 不満
- 명사+**的**(の-な형용사化): 経済**的**, 効果**的**, 積極**的**
- 명사+**中**: 「電話**中**です」(통화 중)
- 명사+**ぶり**: 「**三年ぶり**に会った」 (3년 만에 만났다)

## 예문

1. この方法は**非常識**(ひじょうしき)だ。 — 이 방법은 비상식적이다.
2. **担当者**(たんとうしゃ)に連絡してください。 — 담당자에게 연락해 주세요.
3. 彼の意見は**非常に具体的**(ぐたいてき)だ。 — 그의 의견은 매우 구체적이다.
4. **久しぶり**に故郷に**帰**(かえ)った。 — 오랜만에 고향에 돌아갔다.

## 연습 문제

1. このソフトはまだ___です。
   (A) 未定  (B) 不定  (C) 無定  (D) 非定

2. 田中さんは___です。よく雑誌に出ます。
   (A) 専門員  (B) 専門者  (C) 専門家  (D) 専門人

3. 彼の考え方は非常に___だ。
   (A) 一般  (B) 一般的  (C) 一般性  (D) 一般中

**정답**

1. (A) 未定 — 「아직 정해지지 않음」
2. (C) 専門家 — 전문 분야 사람 (~家)
3. (B) 一般的 — 명사+「的」으로 な형용사화
