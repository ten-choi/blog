## 개요

敬語(경어)는 비즈니스 일본어의 척추. 일본어 경어는 ① **尊敬語(존경어)**: 상대 행위 ↑ 올림, ② **謙譲語(겸양어)**: 내 행위를 ↓ 낮춤, ③ **丁寧語(정중어)**: ます·です 정중체 — 3종으로 나뉜다. N3는 ①②를 본격 다룬다.

## 핵심 문법

### 1. 경어 패턴 정리

| 동사 | 尊敬語(존경) | 謙譲語(겸양) |
|---|---|---|
| する | なさる | いたす |
| 行く / 来る / いる | いらっしゃる / おいでになる | 参(まい)る / おる |
| 言う | おっしゃる | 申(もう)す / 申し上げる |
| 見る | ご覧(らん)になる | 拝見(はいけん)する |
| 食べる / 飲む | 召(め)し上(あ)がる | いただく |
| 聞く / 訪(たず)ねる | (お聞きになる) | 伺(うかが)う |
| 知る | ご存(ぞん)じ | 存じる / 存じ上げる |
| 会う | (お会いになる) | お目にかかる |
| 与える / くれる | くださる | さしあげる |
| もらう | (お受け取りになる) | いただく / 頂戴(ちょうだい)する |

### 2. 일반 동사를 경어로 만드는 공식

- **존경어 (お/ご + 어간 + になる)**: 「先生が**お**書き**になる**」 (선생님이 쓰시다)
- **존경어 (~れる/~られる)**: 「先生が**書かれる**」 (수동형과 동일한 형태)
- **겸양어 (お/ご + 어간 + する/いたす)**: 「私が**お**書き**します**」 (제가 씁니다)
- **3그룹 する동사**: 「**ご**説明**する**」 「**ご**案内**する**」

| 한자어 동사 | 존경 | 겸양 |
|---|---|---|
| 説明する | ご説明になる / 説明される | ご説明する / ご説明いたす |
| 案内する | ご案内になる | ご案内する / ご案内いたす |
| 連絡する | ご連絡になる | ご連絡する / ご連絡いたします |

### 3. 「いらっしゃる」「おっしゃる」 등 불규칙 ます形

- いらっしゃる → いらっしゃい**ます** (い→り로 변화 ×, 단순히 어간 + ます)
- おっしゃる → おっしゃい**ます**
- くださる → ください**ます**
- なさる → なさい**ます**
- ござる → ござい**ます**

### 4. 호칭·접두사 경어

- **お + 일본 고유어**: お名前, お時間, お仕事
- **ご + 한자어**: ご住所(じゅうしょ), ご家族, ご連絡
- **예외**: お電話 (한자어지만 お) — 빈출 단어는 외워두자

### 5. 비즈니스 경어 정형문

- 「**お世話になっております**」 — 잘 부탁드립니다 (정형 인사)
- 「**かしこまりました / 承知しました**」 — 알겠습니다
- 「**少々お待ちください**」 — 잠시만 기다려 주십시오
- 「**よろしくお願いいたします**」 — 잘 부탁드립니다
- 「**~させていただきます**」 — ~하겠습니다 (Unit 11 참고)

## JLPT N3 빈출 패턴

- 「先生は**いらっしゃい**ますか」 → 「います」의 존경
- 「私が**ご案内します**」 → 「案内する」의 겸양
- 「お名前を**伺っても**いいですか」 → 「聞く」의 겸양
- 「**召し上がって**ください」 → 「食べる/飲む」의 존경

## 예문

1. 部長は今、会議室に**いらっしゃい**ます。 — 부장님은 지금 회의실에 계십니다. (존경)
2. 私が**ご案内いたします**。 — 제가 안내해 드리겠습니다. (겸양)
3. もう一度、お名前を**伺っても**よろしいですか。 — 다시 한 번 성함을 여쭤도 될까요? (겸양)
4. どうぞ**召し上がって**ください。 — 어서 드세요. (존경)

## 연습 문제

1. 社長は今、___か。
   (A) いますか  (B) いらっしゃいますか  (C) おりますか  (D) いきますか

2. 「私が___」を겸양어로 가장 자연스러운 것은?
   (A) お説明します  (B) ご説明します  (C) 説明されます  (D) 説明になります

3. ご飯を___ください。
   (A) いただいて  (B) 食べして  (C) 召し上がって  (D) 拝見して

**정답**

1. (B) いらっしゃいますか — 「います」의 존경어
2. (B) ご説明します — 한자어는 「ご + 어간 + する」
3. (C) 召し上がって — 食べる/飲む의 존경
