## 개요

受身(수동)·使役(사역)·使役受身(사역수동)는 N3 문법의 「3대 변환」. 동사 활용 + 조사(に·を) 변화 + 의미 뉘앙스(피해 수동 등)를 한 묶음으로 익혀야 한다.

## 핵심 문법

### 1. 수동형(受身) 만드는 법

| 그룹 | 변환 |
|---|---|
| 1그룹 | う단 → あ단 + れる: 書く → 書か**れる**, 飲む → 飲ま**れる** |
| 2그룹 | る 삭제 + られる: 食べる → 食べ**られる**, 見る → 見**られる** |
| 3그룹 | する → される, 来る → 来(こ)**られる** |

### 2. 수동의 종류

- **직접 수동**: A는 B에게 ~당했다 — 「私は先生に**ほめられた**」(직접 칭찬받음)
- **간접 수동(피해 수동)**: 제3자의 행위가 「나」에게 영향 — 「雨に**降られて**ぬれた」(비를 맞아 젖었다)
- **소유물 수동**: 「私はカバンを**盗まれた**」 (가방을 도둑맞았다)
- **무생물 주어 수동**: 사물·사실 진술 — 「この本は1900年に**書かれた**」(쓰여졌다)

### 3. 사역형(使役) 만드는 법

| 그룹 | 변환 |
|---|---|
| 1그룹 | う단 → あ단 + せる: 書く → 書か**せる**, 飲む → 飲ま**せる** |
| 2그룹 | る 삭제 + させる: 食べる → 食べ**させる** |
| 3그룹 | する → させる, 来る → 来(こ)**させる** |

### 4. 사역의 의미와 조사

- **강제** (A**가** B**에게** ~시키다): 「先生は**学生に**漢字**を**書か**せた**」
- **허가·방임**: 「親は子供**を**遊ば**せる**」 (자동사일 때는 を, 타동사일 때는 に)
- **감정 유발**: 「彼は私**を**困ら**せた**」 (그가 나를 곤란하게 했다)

자동사 사역: 「を」 사용 — 子供**を**泣**かせる**
타동사 사역: 「に」 사용 — 子供**に**本**を**読**ませる**

### 5. 사역수동(使役受身)

- 의미: 「(누가) 시켜서 ~하다」 = 본의 아니게 ~하다·하게 됨
- 만드는 법: **사역형의 어간 + (ら)れる**

| 그룹 | 변환 |
|---|---|
| 1그룹 | 書かせる → 書か**される** (=書かせられる, 단축형) |
| 2그룹 | 食べさせる → 食べ**させられる** |
| 3그룹 | される / 来(こ)**させられる** |

- 단축형 주의: 1그룹의 「~される」는 す로 끝나는 동사에는 안 씀 (話す → 話さ**せられる**, 話さされる ×)

### 6. 의미 뉘앙스 (사역수동)

- 「私は母**に**野菜**を**食べ**させられた**」 — 어머니 때문에 (싫지만) 야채를 먹게 됐다
- 「妹に1時間も待**たされた**」 — 여동생 때문에 1시간이나 기다리게 됨

## JLPT N3 빈출 패턴

- 「~に~を**~(ら)せる**」: 사역
- 「~に**~(ら)れる**」: 수동
- 「~に**~(ら)せられる / ~される**」: 사역수동
- 「**ほめられる / しかられる / 怒られる**」: 감정·평가 수동 빈출

## 예문

1. 子供のとき、母にピアノを**習わされた**。 — 어렸을 때, 엄마 때문에 피아노를 배웠다. (사역수동)
2. 田中さんは先生に**しかられた**。 — 다나카씨는 선생님께 혼났다. (수동)
3. 上司は私**に**残業**を**さ**せた**。 — 상사는 나에게 야근을 시켰다. (사역)
4. 雨**に降られて**、服がぬれた。 — 비를 맞아서 옷이 젖었다. (피해 수동)

## 연습 문제

1. 私は弟に新しいゲームを___。
   (A) 壊した  (B) 壊された  (C) 壊させた  (D) 壊させられた

2. 先生は学生に作文を___。
   (A) 書いた  (B) 書かれた  (C) 書かせた  (D) 書かされた

3. 子供のとき、毎日ピアノを___て大変だった。
   (A) 練習した  (B) 練習された  (C) 練習させた  (D) 練習させられ

**정답**

1. (B) 壊された — 동생이 부숨 → 내가 피해 (수동)
2. (C) 書かせた — 선생님이 시킴 (사역)
3. (D) 練習させられ — 시켜서 어쩔 수 없이 했다 (사역수동)
