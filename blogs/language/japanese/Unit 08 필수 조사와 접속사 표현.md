## 개요

N3 文法(문법) 영역의 출발점. 조사(助詞)는 의미를 결정짓는 「방향타」, 접속표현(接続表現)은 문장 흐름의 「길잡이」. 단편적 외움이 아니라 **호응 동사**·**전후 문맥**과 묶어 익혀야 한다.

## 핵심 문법

### 1. 기본 격조사(格助詞) 점검

| 조사 | 주요 의미 |
|---|---|
| が | 주격, 신정보, 가능·능력의 대상 |
| を | 직접목적어, 이동의 통과 (公園を歩く) |
| に | 도착점, 시간, 상대, 변화 결과 |
| で | 장소, 수단, 원인 |
| と | ~과 함께, 인용 |
| から | 출발점, 원인, 재료(눈에 안 보임) |
| まで | ~까지 (한계) |
| より | ~보다 (비교) |

- 「学校**で**勉強する」(장소) vs 「学校**に**行く」(도착점)
- 「ワイン**から**作る」(눈에 안 보이는 변화) vs 「木**で**作る」(재료)

### 2. N3 핵심 부조사(副助詞)·계조사(係助詞)

| 조사 | 의미·용법 |
|---|---|
| は | 주제 제시, 대비 |
| も | ~도 |
| しか~ない | ~밖에 없다 |
| だけ | ~만 (한정) |
| ばかり | ~만 (편중), 갓 ~한 |
| こそ | ~야말로 |
| さえ | ~조차 |
| でも | ~라도 |
| なんか / なんて | ~따위 / ~따위라니 |

### 3. 인과·이유 접속표현

| 표현 | 뉘앙스 |
|---|---|
| から | 주관적 이유, 회화체 |
| ので | 객관적 이유, 정중함 |
| ために | (목적) ~을 위해 / (이유) 그래서 |
| せいで | 부정적 원인 (탓에) |
| おかげで | 긍정적 원인 (덕분에) |
| ものだから | 변명·핑계 |

### 4. 역접·대비 접속표현

| 표현 | 뉘앙스 |
|---|---|
| が / けど / けれど(も) | 일반 역접 |
| のに | ~인데도 (의외·불만) |
| ても / でも | ~해도 |
| くせに | ~인 주제에 (부정적) |
| 반면(反面) | ~한 반면 |
| 一方(いっぽう)で | 한편으로 |

### 5. 첨가·순서 접속표현

- それに / さらに / その上(うえ) — 게다가
- それから / そして / さらに — 그리고
- まず / 最初に / 始めに — 우선
- 次に / その次に — 다음에
- 最後に / 最終的に — 마지막에
- ところで — 그런데 (화제 전환)
- というのは — 왜냐하면

## JLPT N3 빈출 패턴

- 「~を通(つう)じて / ~を通(とお)して」: ~을 통해
- 「~について」: ~에 대해
- 「~に対(たい)して」: ~에 대해 (대상·반대)
- 「~に関(かん)して」: ~에 관해
- 「~とともに」: ~와 함께
- 「~として」: ~로서

## 예문

1. 雨**のに**、傘を持ってこなかった。 — 비가 오는데도 우산을 안 가져왔다.
2. お金がない**から**、買えない。 — 돈이 없어서 살 수 없다.
3. インターネット**を通じて**いろいろな情報を得る。 — 인터넷을 통해 다양한 정보를 얻는다.
4. 彼**だけ**が真実(しんじつ)を知っている。 — 그만이 진실을 알고 있다.

## 연습 문제

1. 雨が降っている___、傘を忘れた。
   (A) ので  (B) のに  (C) から  (D) ために

2. 先生___この問題について説明した。
   (A) は  (B) を  (C) に対して  (D) と

3. このカメラは小さい。___、軽い。
   (A) その上  (B) しかし  (C) けれど  (D) ところで

**정답**

1. (B) のに — 의외·불만의 역접
2. (C) に対して — 「~에 대해 설명」
3. (A) その上 — 첨가 (게다가)
