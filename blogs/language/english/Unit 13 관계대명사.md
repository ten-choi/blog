## 개요

관계대명사는 **선행사(사람/사물)**와 **관계절 안에서의 역할(주어/목적어/소유격)**에 따라 결정된다. 자리만 잡으면 답은 한 가지로 정해진다.

## 핵심 문법

### 1. 관계대명사 선택표

| 선행사 | 주격 | 목적격 | 소유격 |
|---|---|---|---|
| 사람 | who / that | whom / who / that | whose |
| 사물·동물 | which / that | which / that | whose / of which |
| 사람 + 사물 | that | that | — |

### 2. 격 판별법

빈칸 뒤를 본다:
- 뒤에 **동사** → 주격 (who, which, that)
- 뒤에 **주어 + 동사** → 목적격 (whom, which, that — 생략 가능)
- 뒤에 **명사** → 소유격 (whose)

### 3. 예시

- 주격: *The manager **who** leads the project is new.*
- 목적격: *The report **(that/which)** we submitted was approved.* (생략 가능)
- 소유격: *The employee **whose** ID was lost contacted security.*

### 4. 관계부사

선행사가 시간/장소/이유/방법일 때.

| 선행사 | 관계부사 | 풀어쓰면 |
|---|---|---|
| 시간 (the day, the time) | when | at/on which |
| 장소 (the place, the office) | where | in/at which |
| 이유 (the reason) | why | for which |
| 방법 (the way) | how (단, the way와 함께 못 씀) | in which |

- *the office **where** we work* = the office **in which** we work

### 5. 제한적 용법 vs 계속적 용법

- **제한적**: 콤마 X — 선행사 의미 한정
  - *Employees **who work** overtime get bonuses.* (오버타임 근무하는 직원만)
- **계속적**: 콤마 O — 부연 설명, **that 사용 불가**
  - *Ms. Lee, **who** is the team leader, will attend.*

### 6. 전치사 + 관계대명사

- *the project **on which** we are working* = the project **(that)** we are working **on**
- 격식 있는 문체에서 전치사 + which / whom 사용
- 단, **that 앞에는 전치사 불가** — ✗ on that

### 7. 자주 쓰이는 패턴

- **those who** ~ : ~하는 사람들
- **anyone who** ~ : ~하는 누구든
- **what** (선행사 포함 관계대명사): the thing(s) which — *What I need is time.*

## TOEIC 빈출 패턴

- `명사 + _____ + 동사` → 주격 (who/which/that)
- `명사 + _____ + 주어 + 동사` → 목적격
- `명사 + _____ + 명사 + 동사` → 소유격 (whose)
- `장소/시간 명사 + _____ + 주어 + 동사` → 관계부사 (where/when)

## 예문

1. The employee **who** received the award gave a speech.
2. The proposal **that** we submitted was accepted.
3. The conference room **where** we meet is on the 5th floor.
4. Applicants **whose** experience matches the role will be interviewed.

## 연습 문제

1. The candidate _____ we interviewed yesterday accepted the offer.
   (A) who  (B) whom  (C) whose  (D) which

2. The factory _____ produces our main products is in Vietnam.
   (A) who  (B) which  (C) where  (D) whose

3. We hired a consultant _____ expertise is in marketing.
   (A) who  (B) which  (C) whose  (D) whom

**정답**

1. (B) whom — 사람 선행사, 뒤에 we interviewed (목적격)
2. (B) which — 사물 선행사 (factory), 뒤에 동사 produces (주격)
3. (C) whose — 뒤에 명사 expertise (소유격)
