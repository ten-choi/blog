## 개요

형용사는 **명사 수식** 또는 **보어** 자리에 들어간다. 토익에서는 비슷하게 생긴 단어(형용사/명사/부사)를 보기에 섞어 놓고 자리만으로 답을 가르는 문제가 흔하다.

## 핵심 문법

### 1. 형용사 자리

- **명사 앞**: a *reliable* partner / *new* policy
- **보어 자리** (be동사·become·seem·remain 등 뒤): The system is *reliable*.
- **목적격 보어**: We found the system *reliable*.

### 2. 형용사 어미

| 어미 | 예 |
|---|---|
| -ful | helpful, careful, successful |
| -less | useless, careless |
| -ous | dangerous, previous, various |
| -al | financial, additional, professional |
| -ive | effective, productive, competitive |
| -able / -ible | reliable, accessible, available |
| -ent / -ant | different, important, relevant |
| -y | busy, easy, healthy |

### 3. 혼동 형용사

| 단어 | 의미 |
|---|---|
| economic | 경제의 (economic growth) |
| economical | 경제적인, 절약하는 (economical car) |
| considerable | 상당한 |
| considerate | 사려 깊은 |
| respectable | 존경할 만한 |
| respective | 각각의 |
| successful | 성공적인 |
| successive | 연속적인 |
| sensible | 분별 있는 |
| sensitive | 민감한 |

### 4. 수량 형용사 (가산/불가산 구분)

| 가산 명사와 | 불가산 명사와 | 둘 다 |
|---|---|---|
| many, few, a few, several | much, little, a little | some, any, a lot of, plenty of |

- *many employees* / *much information* / *a lot of products*

## TOEIC 빈출 패턴

- `_____ + 명사`: 형용사 (extensive experience)
- `be + _____`: 형용사 보어 (be available, be responsible)
- `several / various / numerous + 복수명사`: 양·다양성 강조
- `available / responsible / eligible + for / to V`: 전치사·to부정사 동반

## 예문

1. We offer **competitive** salaries to attract top talent.
2. The new policy is **effective** immediately.
3. Please provide **detailed** feedback on the proposal.
4. The position is currently **available**.

## 연습 문제

1. The company is looking for an _____ candidate for the role.
   (A) experience  (B) experienced  (C) experiencing  (D) experiences

2. Our service has been _____ to all customers.
   (A) help  (B) helpful  (C) helpfully  (D) helps

3. The deadline is _____ tomorrow.
   (A) extension  (B) extended  (C) extending  (D) extensions

**정답**

1. (B) experienced — 관사 an + 형용사 + 명사
2. (B) helpful — be + 형용사 보어
3. (B) extended — be + 과거분사 (수동, 형용사 역할)
