## 개요

부사절 접속사는 의미별로 묶어 외우는 게 효율적. 토익에서는 **의미가 비슷한 접속사 간 구분**(although vs because), 그리고 **전치사구와의 구분**이 핵심.

## 핵심 문법

### 1. 의미별 부사절 접속사

| 의미 | 접속사 |
|---|---|
| 시간 | when, while, before, after, until, as soon as, once, by the time, as |
| 조건 | if, unless, provided (that), as long as, in case |
| 이유 | because, since, as, now that |
| 양보 | although, though, even though, even if, while, whereas |
| 목적 | so that, in order that |
| 결과 | so + 형용사/부사 + that, such + 명사 + that |

### 2. 시간 / 조건 부사절: 미래는 현재로

- *When the report **arrives**, we'll review it.* (○ arrives, ✗ will arrive)
- *If you **need** help, let me know.* (○ need)

### 3. 양보 (~에도 불구하고)

- **although / though / even though** + 주어 + 동사
- 회화에서 though는 문장 끝에 부사로도 쓰임 — *It's expensive, though.*

### 4. 목적 vs 결과

- **목적 (so that)**: 주어 + 동사 + so that + 주어 + can/will + V
  - *We arrived early **so that** we **could prepare**.*
- **결과 (so ~ that)**: so + 형용사/부사 + that + 절
  - *The proposal was **so detailed that** everyone was convinced.*
- **결과 (such ~ that)**: such + (관사) + 형용사 + 명사 + that + 절
  - *It was **such a successful event that** we'll repeat it.*

### 5. 자주 쓰이는 부사절 접속사

- **as soon as**: ~하자마자
- **by the time**: ~할 때쯤
- **now that**: 이제 ~하니까 (이유)
- **provided (that) / as long as**: ~하기만 하면 (조건)
- **whereas / while**: 반면에 (대조)
- **in case**: ~에 대비해서

### 6. 분사구문 vs 부사절

- 부사절: *When she finished, she left.*
- 분사구문: *Finishing, she left.*
- 토익은 둘 다 자주 출제 — 빈칸이 절을 이끄는 자리인지 분사구문 자리인지 형태로 구분.

## TOEIC 빈출 패턴

- `_____ + 주어 + 동사, 주어 + 동사` → 부사절 접속사
- `Although vs Despite`: 뒤에 절이면 Although, 명사구면 Despite
- `so + adj + that` vs `such + 명사 + that` 구분
- `not until + 절, 조동사 + 주어 + 동사 원형` (도치 — Unit 15 참조)

## 예문

1. **Although** the project was challenging, the team finished on time.
2. The team worked overtime **so that** they could meet the deadline.
3. **As soon as** the report is ready, please email it to me.
4. **Now that** the contract is signed, we can begin production.

## 연습 문제

1. _____ the weather is bad, the outdoor event will be postponed.
   (A) Despite  (B) If  (C) Whereas  (D) Because of

2. The conference was _____ informative that we extended it.
   (A) so  (B) such  (C) very  (D) too

3. _____ the deadline passes, we cannot accept any applications.
   (A) Once  (B) During  (C) Despite  (D) For

**정답**

1. (B) If — 뒤에 절 (the weather is bad), 조건 부사절
2. (A) so — so + 형용사 + that (informative는 형용사)
3. (A) Once — "일단 ~하면", 시간·조건 부사절 접속사
