## 개요

부사는 **명사 외 모든 것**을 수식한다 — 동사, 형용사, 다른 부사, 문장 전체. 토익에서는 동사 앞뒤·형용사 앞·be동사와 분사 사이 빈칸이 부사 단골 자리.

## 핵심 문법

### 1. 부사 자리

- **동사 수식**: works *efficiently*, *carefully* reviewed
- **형용사 수식**: *highly* competitive
- **부사 수식**: *very* quickly
- **문장 전체 수식**: *Fortunately*, the deal closed.
- **be + 부사 + 과거분사**: was *successfully* completed
- **have + 부사 + 과거분사**: has *recently* joined

### 2. 부사 어미

대부분 **형용사 + -ly**: quickly, carefully, recently, successfully

예외 (형용사와 형태 같음): **fast, hard, late, early, near, high, long**

`-ly`가 붙어도 형용사인 단어: **friendly, lovely, costly, timely, weekly, monthly**

### 3. 혼동 부사

| 단어 | 의미 |
|---|---|
| hard | 열심히 |
| hardly | 거의 ~않는 |
| late | 늦게 |
| lately | 최근에 |
| high | 높이 |
| highly | 매우 |
| near | 가까이 |
| nearly | 거의 |
| most | 대부분, 가장 |
| mostly | 주로, 대부분 |

### 4. 자주 쓰이는 부사

- **시간**: currently, recently, soon, already, still, yet
- **빈도**: always, usually, often, sometimes, rarely, never
- **정도**: very, quite, fairly, extremely, highly, completely
- **부정 강조**: hardly, scarcely, rarely, seldom (부정문 표현 자체로 사용)

### 5. still / already / yet

- **still**: 아직도 (긍정문) — *He is still working.*
- **already**: 이미 (긍정문) — *She has already left.*
- **yet**: 아직 (부정·의문문) — *Has the package arrived yet?*

## TOEIC 빈출 패턴

- `be + _____ + p.p.`: 부사 (was carefully reviewed)
- `_____ + 형용사`: 부사 (highly recommended)
- `동사 + 목적어 + _____`: 부사 (handled the issue *promptly*)
- 빈출 부사: **currently, recently, immediately, successfully, significantly, carefully, completely**

## 예문

1. The proposal was **carefully** reviewed by the board.
2. We are **currently** hiring for several positions.
3. The new system is **highly** efficient.
4. She has **recently** been promoted to manager.

## 연습 문제

1. The team _____ completed the project ahead of schedule.
   (A) success  (B) successful  (C) successfully  (D) succeeds

2. Sales have grown _____ since last quarter.
   (A) significant  (B) significance  (C) significantly  (D) signify

3. The CEO is _____ pleased with the results.
   (A) high  (B) highly  (C) higher  (D) highest

**정답**

1. (C) successfully — 동사 completed 수식, 부사 자리
2. (C) significantly — 자동사 grown 수식
3. (B) highly — 형용사 pleased 수식 (high는 "높이"의 의미라 부적합)
