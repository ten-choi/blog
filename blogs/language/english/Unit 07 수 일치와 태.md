## 개요

수 일치는 **주어를 정확히 찾는 것**이 절반. 주어와 동사 사이에 긴 수식어가 끼어 있으면 진짜 주어가 무엇인지부터 확인한다. 태는 **목적어 유무**로 능동/수동을 판단.

## 핵심 문법

### 1. 주어-동사 수 일치 원칙

- 단수 주어 + 단수 동사 / 복수 주어 + 복수 동사
- 주어와 동사 사이의 **수식어구는 무시**:
  - The *manager*, along with two assistants, **is** attending.
  - The *reports* on the desk **are** outdated.

### 2. 주의해야 할 주어

| 형태 | 일치 |
|---|---|
| each / every / one of + 복수 명사 | 단수 |
| some / most / all / half + of + 단수 | 단수 |
| some / most / all / half + of + 복수 | 복수 |
| the number of + 복수 명사 | 단수 ("수") |
| a number of + 복수 명사 | 복수 ("다수의") |
| both A and B | 복수 |
| either A or B / neither A nor B | B에 일치 |
| not only A but also B | B에 일치 |

### 3. 능동태 vs 수동태

- **능동**: 주어가 행위 주체 — *The team completed the project.*
- **수동**: 주어가 행위 대상 — *The project was completed by the team.*

수동태 = **be + 과거분사 (p.p.)**

### 4. 능동/수동 판별법

빈칸 뒤를 본다:
- 빈칸 뒤에 **목적어(명사)** 있으면 → **능동**
- 빈칸 뒤에 **목적어 없음** (전치사구·부사로 끝) → **수동**

### 5. 수동태가 자주 쓰이는 동사

- be **made / produced / manufactured**
- be **delivered / shipped / sent**
- be **scheduled / postponed / canceled**
- be **required / expected / encouraged**
- be **based on / known for / interested in** (수동 + 전치사 묶음)

### 6. 4형식·5형식의 수동태

- 4형식: 직접목적어/간접목적어 둘 다 주어 가능
  - *I was given the report.* / *The report was given to me.*
- 5형식: 보어는 그대로
  - *He was asked to attend.* (ask + O + to V → be asked to V)

## TOEIC 빈출 패턴

- `The list of attendees _____ on the desk.` → is (주어 = list, 단수)
- `Each of the employees _____ trained.` → was (each → 단수)
- `The contract _____ by both parties yesterday.` → was signed (목적어 없음 + by → 수동)
- `The proposal _____ several concerns.` → addresses (목적어 있음 → 능동)

## 예문

1. The CEO, along with the executives, **is** attending the conference.
2. A number of employees **have** signed up for the training.
3. The package **was delivered** to the wrong address.
4. All staff **are required to** submit timesheets weekly.

## 연습 문제

1. The list of all attendees _____ posted on the bulletin board.
   (A) is  (B) are  (C) were  (D) have been

2. The contract _____ by both parties last Friday.
   (A) signed  (B) was signed  (C) signing  (D) is signing

3. Each of the candidates _____ a personal interview.
   (A) receive  (B) receives  (C) have received  (D) are receiving

**정답**

1. (A) is — 주어는 list (단수), of all attendees는 수식어
2. (B) was signed — 뒤에 목적어 없고 by 행위자 → 수동
3. (B) receives — each of + 복수 → 단수 동사
